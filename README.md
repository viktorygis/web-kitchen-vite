# web-kitchen-vite

Многостраничный статический сайт (MPA) на **Vite**.

- HTML-страницы лежат в `src/` (и подпапках).
- Общие фрагменты подключаются через `@@include()` из `src/partials/`.
- Стили — **SCSS** (`src/styles/`).
- Скрипты — модульные **ESM** (`src/scripts/`).

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте: `http://localhost:5173/`

## Скрипты

- `npm run dev` — режим разработки
- `npm run build` — production-сборка в `dist/`
- `npm run preview` — предпросмотр сборки

## Структура

```text
.
├─ public/                # статика (img, fonts)
├─ scripts/               # vite-плагин для @@include()
├─ src/
│  ├─ index.html
│  ├─ *.html              # корневые страницы
│  ├─ */**/*.html         # страницы в подпапках
│  ├─ partials/           # НЕ страницы, а общие фрагменты
│  ├─ styles/             # SCSS
│  └─ scripts/            # JS
└─ vite.config.js
```

## @@include()

Пример для корневой страницы:

```html
@@include('partials/head.html', {"title": "Главная", "description": "..."})
```

Пример для страницы в подпапке:

```html
@@include('../partials/head.html', {"title": "Страница", "description": "..."})
```

## Деплой

- Frontend: выполните `npm run build` и загрузите содержимое `dist/` на хостинг.
