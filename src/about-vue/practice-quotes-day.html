Описание проекта

Это простое веб-приложение на Vue.js, которое позволяет пользователям получать случайные цитаты дня. При нажатии на кнопку "Получить цитату" приложение отправляет запрос к публичному API, получает случайную цитату и отображает её на странице. Если возникает ошибка при запросе, пользователю показывается сообщение об ошибке.

Проект демонстрирует использование Vue.js для создания интерактивного пользовательского интерфейса, а также работу с Vuex для управления состоянием приложения. Это отличный пример того, как можно быстро создать полезное приложение с минимальными усилиями.

Код HTML

<div id="app" class="app">
  <!-- Заголовок приложения -->
  <h1>Цитаты дня</h1>

  <!-- Вступительное сообщение, показывается до первого клика -->
  <p class="intro-message" v-if="!hasClicked">Чего ты ждешь? Жми скорее, чтобы получить цитату 👇</p>

  <!-- Текущая цитата или сообщение об ошибке, показываются после первого клика -->
  <p class="quote" v-if="hasClicked">{{ currentQuote }}</p>

  <!-- Кнопка для получения новой цитаты -->
  <button @click="fetchQuote">Получить цитату</button>

  <!-- Сообщение о загрузке -->
  <div v-if="loading" class="loading">Загружаем новую цитату...</div>
</div>

Описание кода html

Этот HTML-код представляет собой структуру веб-приложения на Vue.js для отображения цитат дня. Вот описание основных частей кода:

  • Корневой элемент: 
    <div id="app" class="app">
    - это корневой элемент приложения Vue.js, который связывается с экземпляром Vue и содержит все элементы интерфейса.
  • Заголовок: 
    <h1>Цитаты дня</h1>
    - заголовок приложения, который информирует пользователя о его назначении.
  • Вступительное сообщение: 
    <p class="intro-message" v-if="!hasClicked">
  Чего ты ждешь? Жми скорее, чтобы получить цитату 👇
</p>
    - параграф, который отображается только до тех пор, пока пользователь не нажмет кнопку для получения цитаты. Он побуждает пользователя к действию.
    Где v-if="!hasClicked" - условие, которое проверяет, был ли клик по кнопке.
  • Текущая цитата или сообщение об ошибке: 
    <p class="quote" v-if="hasClicked">{{ currentQuote }}</p>
    - параграф, который отображает цитату или сообщение об ошибке после первого клика.
    Где v-if="hasClicked" - условие, которое проверяет, был ли клик по кнопке, а {{ currentQuote }} - это привязка данных Vue.js, которая отображает текущее значение цитаты из состояния приложения.
  • Кнопка для получения новой цитаты: 
    <button @click="fetchQuote">Получить цитату</button>
    - кнопка, при нажатии на которую вызывается метод fetchQuote, который инициирует процесс получения новой цитаты.
    @click=""> - событие клика по кнопке, которое вызывает метод fetchQuote.
  • Сообщение о загрузке: 
    <div v-if="loading" class="loading">Загружаем новую цитату...</div>
    - блок, который отображается, когда приложение находится в состоянии загрузки новой цитаты, информируя пользователя о процессе.

Код JavaScript

//Инициализация Vuex хранилища
const store = Vuex.createStore({
  //Состояние приложения
  state() {
    return {
      //currentQuote - текущая цитата, loading - состояние загрузки
      currentQuote: "",
      loading: false,
    };
  },
  //Мутации для изменения состояния
  mutations: {
    setQuote(state, quote) {
      state.currentQuote = quote;
    },
    setLoading(state, status) {
      state.loading = status;
    },
  },
  //Асинхронные действия
  actions: {
    async fetchQuote({ commit }) {
      commit("setLoading", true);
      try {
        const response = await fetch("https://dummyjson.com/quotes");
        if (!response.ok) throw new Error("Ошибка API");
        const data = await response.json();
        const randomQuote = data.quotes[Math.floor(Math.random() * data.quotes.length)];
        commit("setQuote", randomQuote.quote);
      } catch (error) {
        console.error("Ошибка запроса:", error.message);
        commit("setQuote", "Сетевая ошибка! Проверьте подключение и попробуйте снова.");
      } finally {
        commit("setLoading", false);
      }
    },
  },
});

//Создание экземпляра Vue.js
const app = Vue.createApp({
  //Данные компонента
  data() {
    //Возвращаемое значение
    return {
      //hasClicked отслеживает, был ли клик по кнопке
      hasClicked: false,
    };
  },
  //Вычисляемые свойства
  computed: {
    //Подключение состояния из Vuex хранилища
    //currentQuote - текущая цитата, loading - состояние загрузки
    ...Vuex.mapState(["currentQuote", "loading"]),
  },

  //Методы компонента
  methods: {
    //Метод для получения новой цитаты
    fetchQuote() {
      this.$store.dispatch("fetchQuote");
      this.hasClicked = true;
    },
  },
});

app.use(store);
app.mount("#app");

Описание кода script

Этот JavaScript-код создает веб-приложение на Vue.js с использованием Vuex для управления состоянием. Вот описание основных частей кода:

  • Инициализация Vuex хранилища: 
    const store = Vuex.createStore({ ... });
    - создается хранилище Vuex, которое управляет состоянием приложения.
  • Состояние приложения: 
    state() { return { currentQuote: "", loading: false, }; }
    - определяет начальное состояние приложения с двумя свойствами: currentQuote для хранения текущей цитаты и loading для отслеживания состояния загрузки.
  • Мутации для изменения состояния: 
    mutations: { setQuote(state, quote) { ... }, setLoading(state, status) { ... }, }
    - определяет методы для изменения состояния хранилища. setQuote обновляет текущую цитату, а setLoading изменяет состояние загрузки.
  • Асинхронные действия: 
    actions: { async fetchQuote({ commit }) { ... }, }
    - определяет асинхронное действие fetchQuote, которое отправляет запрос к API для получения случайной цитаты. В случае успеха цитата сохраняется в состоянии, а в случае ошибки отображается сообщение об ошибке.
  • Создание экземпляра Vue.js: 
    const app = Vue.createApp({ ... });
    - создает экземпляр Vue.js, который управляет пользовательским интерфейсом приложения.
  • Данные компонента: 
    data() { return { hasClicked: false, }; }
    - определяет локальное состояние компонента с переменной hasClicked, которая отслеживает, был ли клик по кнопке для получения цитаты.
  • Вычисляемые свойства: 
    computed: { ...Vuex.mapState(["currentQuote", "loading"]), }
    - подключает состояние из Vuex хранилища к компоненту, позволяя использовать currentQuote и loading в шаблоне.
  • Методы компонента: 
    methods: { fetchQuote() { ... }, }
    - определяет метод fetchQuote, который вызывает действие Vuex для получения новой цитаты и устанавливает hasClicked в true после первого клика.
