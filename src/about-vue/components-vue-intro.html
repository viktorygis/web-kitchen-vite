Компоненты во Vue

Компоненты — основа любого современного приложения на Vue.
Это независимые, переиспользуемые куски кода — «строительные блоки» интерфейса.
Они позволяют разделять приложение на логические части, упрощать разработку и повторное использование.

  • Изолируют логику и шаблон
  • Упрощают структуру приложения
  • Позволяют повторно использовать код

Регистрация компонентов

Регистрация компонента — процесс объявления компонента, чтобы Vue знал о его существовании и мог его использовать.

Виды регистрации компонента:

  • Локальная регистрация
  • Глобальная регистрация

Локальная регистрация

Локальная регистрация - компонент доступен только внутри родительского компонента, в котором он зарегистрирован.

Пример локальной регистрации компонента:

// Создание приложения Vue
const app = Vue.createApp({
  components: {
    'hello-button': {
      template: '<button>Привет</button>'
    }
  }
});
app.mount('#local-demo');

// Использование компонента в шаблоне
app.template = '<hello-button></hello-button>';
Объяснение кода:
  • components — опция, где регистрируются локальные компоненты.
  • 'hello-button' — имя компонента, используемое в шаблоне.
  • template — HTML-шаблон компонента.

Глобальная регистрация

Глобальная регистрация — компонент можно использовать в любом месте приложения.

Пример глобальной регистрации компонента:

//Создание приложения Vue
const app = Vue.createApp({});
app.component('global-button', {
  template: '<button>Глобальная кнопка</button>'
});
app.mount('#global-demo');

//Использование компонента в шаблоне
app.template = '<global-button></global-button>';
Объяснение кода:
  • app.component — метод для глобальной регистрации компонента.
  • 'global-button' — имя компонента, доступное во всем приложении.
  • template — HTML-шаблон компонента.

Различие между локальной и глобальной регистрацией

  • Локальная регистрация ограничивает область видимости компонента родительским компонентом, что помогает избежать конфликтов имен и улучшает организацию кода.
  • Глобальная регистрация делает компонент доступным в любом месте приложения, что удобно для часто используемых компонентов, но может привести к конфликтам имен.

Создание компонента во Vue

Создание компонента во Vue включает в себя определение шаблона, логики и стилей компонента. Компоненты могут быть зарегистрированы локально или глобально, в зависимости от того, где они будут использоваться. Это позволяет эффективно управлять структурой приложения и повторно использовать код.

Шаги создания компонента во Vue:

  1. Определение шаблона: Создайте HTML-шаблон, который определяет структуру и внешний вид компонента.
    Это можно сделать с помощью тега <template>.
    Пример:
    <template>
  <h1>Заголовок компонента</h1>
</template>
  2. Добавление логики: Определите поведение компонента с помощью JavaScript, включая данные, методы и жизненный цикл.
    Это обычно делается внутри тега <script>.
    Пример:
    <script>
export default {
  name: "MyComponent",
  data() {
    return {
      message: "Привет, мир!",
    };
  },
  methods: {
    greet() {
      alert(this.message);
    },
  },

};
</script>
  3. Регистрация компонента: Зарегистрируйте компонент локально или глобально, чтобы использовать его в приложении.
    Локальная регистрация выполняется внутри родительского компонента, а глобальная — с помощью метода app.component.
    Пример:
    // Локальная регистрация
const app = Vue.createApp({
  components: {
    'my-component': MyComponent
  }
});
// Глобальная регистрация
app.component('my-component', MyComponent);

Пример создания компонента PageTitle.vue:

1. Определение компонента:

Создание простого компонента, который отображает заголовок страницы. Для создания компонента создайте файл PageTitle.vue в папке src/components/:

<!-- src/components/PageTitle.vue -->
<template>
<h1>Мой первый компонент</h1>
</template>

<script>
export default {
  name: "PageTitle",
};
 </script>

2. Использование компонента в приложении:

Теперь импортируйте и используйте компонент PageTitle в основном файле приложения, например, в App.vue:

<!-- src/App.vue -->
<script>
import PageTitle from "./components/PageTitle.vue";

export default {
  name: "App",
  components: { PageTitle },
};
</script>

<template>
  <h4>Заголовок с помощью компонента</h4>
  <PageTitle />
</template>
