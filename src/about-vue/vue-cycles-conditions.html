Циклы и условные конструкции во Vue

Циклы и условные конструкции во Vue — ключевые инструменты для динамического отображения данных и управления видимостью элементов на странице.

Циклы и условные конструкции нужны для:

  • Динамического создания списков и элементов на основе данных
  • Управления видимостью контента в зависимости от состояния приложения
  • Создания интерактивных интерфейсов с адаптивным отображением информации

Основные директивы:

  • v-for — директива для перебора массивов и отображения элементов списка.
  • v-if — директива для условного отображения элементов на основе логических выражений.
  • v-else — директива для отображения альтернативного контента, если условие v-if не выполнено.
  • v-else-if — директива для добавления дополнительных условий к цепочке v-if/v-else.
  • v-show — директива для условного отображения элементов с помощью CSS-свойства display.

🚀 Освоение циклов и условных конструкций во Vue позволит создавать динамичные и интерактивные веб-приложения с гибким управлением отображением контента.

v-for

v-for - директива Vue.js для перебора массивов и отображения элементов списка. Она позволяет динамически создавать элементы на основе данных, что делает интерфейсы более гибкими и интерактивными.

Базовый синтаксис:

<li v-for="item in items" :key="item.id">{{ item }}</li>
Объяснение кода:
  • v-for — директива для перебора массива items.
  • item — текущий элемент массива items в каждой итерации.
  • :key — уникальный идентификатор для каждого элемента, помогающий Vue эффективно обновлять DOM.
  • item.id — предполагается, что каждый элемент массива имеет уникальное свойство id.
  • {{ item }} — отображение текущего элемента в списке.

Пример:

Отобразим список пользователей с их именами и возрастом:

  • 👤 {{ user.name }} (Возраст: {{ user.age }}) 
<ul>
   <li v-for="user in users" :key="user.id">
     👤 {{ user.name }} (Возраст: {{ user.age }})
   </li>
</ul>

 <script>
 Vue.createApp({
   data() {
     return {
       users: [
         { id: 1, name: "Алиса", age: 25 },
         { id: 2, name: "Боб", age: 30 },
         { id: 3, name: "Чарли", age: 35 }
       ]
     }
   }
 }).mount("#users-demo");
 </script>
Объяснение кода:
  • Создаём Vue-приложение с массивом пользователей в данных.
  • Используем v-for для перебора массива users и отображения каждого пользователя в списке.
  • Каждому элементу списка присваиваем уникальный ключ с помощью :key="user.id".

Нумерация элементов: v-for с индексом

Нумерация элементов: v-for с индексом позволяет отображать порядковый номер каждого элемента в списке.

Пример с нумерацией пользователей:

Отобразим список пользователей с их именами и порядковыми номерами:

  • {{ index + 1 }}. 👤 {{ user.name }} 
<ul>
  <li v-for="(user, index) in users" :key="user.id">
    {{ index + 1 }}. 👤 {{ user.name }}
  </li>
</ul>
Объяснение кода:
  • Используем синтаксис (user, index) для получения текущего элемента и его индекса.
  • Отображаем порядковый номер с помощью {{ index + 1 }}, так как индексация начинается с 0.

🚀 v-for с индексом — удобный способ нумеровать элементы в списке при их отображении.

Условное отображение

v-if, v-else и v-show — директивы Vue.js для условного отображения элементов на странице. Они позволяют управлять видимостью контента в зависимости от состояния приложения или логических условий.

v-if и v-else

v-if - директива для условного отображения элемента, если выражение истинно.
v-else - директива для отображения альтернативного контента, если условие v-if ложно.

Различие между v-if и v-else:

  • v-if проверяет условие и отображает элемент, если оно истинно.
  • v-else следует сразу за v-if и отображается, когда условие v-if ложно.

Пример:

Отобразим кнопки "Войти" и "Выйти" в зависимости от состояния авторизации пользователя:

<div id="auth-demo">
  <button v-if="isLoggedIn">Выйти</button>
  <button v-else>Войти</button>
</div>

<script>
Vue.createApp({
  data() {
    return { isLoggedIn: false }
  }
}).mount("#auth-demo");
</script>
Объяснение кода:
  • Создаём Vue-приложение с булевым состоянием isLoggedIn.
  • Используем v-if для отображения кнопки "Выйти", если пользователь авторизован.
  • Используем v-else для отображения кнопки "Войти", если пользователь не авторизован.

v-show

v-show - директива для условного отображения элемента с помощью CSS-свойства display. В отличие от v-if, элемент остаётся в DOM, но скрывается визуально.

Отличием между v-if и v-show:

  • v-if добавляет или удаляет элемент из DOM в зависимости от условия.
  • v-show всегда оставляет элемент в DOM, но управляет его видимостью через CSS.

Пример:

Скрыть и показать текст по кнопке:

Этот текст можно скрыть!

<div id="show-demo">
  <button @click="isVisible = !isVisible">Переключить</button>
  <p v-show="isVisible">Этот текст можно скрыть!</p>
</div>

<script>
Vue.createApp({
  data() {
    return { isVisible: true }
  }
}).mount("#show-demo");
</script>
Объяснение кода:
  • Создаём Vue-приложение с булевым состоянием isVisible.
  • Используем v-show для управления видимостью текста по кнопке.
  • Кнопка переключает значение isVisible, скрывая или показывая текст.
Когда использовать v-if vs v-show:
  • Используйте v-if, когда условие редко меняется и важно экономить ресурсы DOM.
  • Используйте v-show, когда нужно часто переключать видимость элемента без удаления из DOM.

v-else-if: несколько условий

v-else-if — директива Vue.js для добавления промежуточных условий между v-if и v-else. Она позволяет создавать цепочки условий для более сложной логики отображения элементов.

Пример:

Покажем разные сообщения в зависимости от роли пользователя:

Привет, Админ! 👑

Привет, Пользователь! 👤

Привет, Гость! 🚪

<div id="role-demo">
  <p v-if="role === 'admin'">Привет, Админ! 👑</p>
  <p v-else-if="role === 'user'">Привет, Пользователь! 👤</p>
  <p v-else>Привет, Гость! 🚪</p>
</div>

<script>
Vue.createApp({
  data() {
    return { role: "user" } // 'admin', 'user', 'guest'
  }
}).mount("#role-demo");
</script>
Объяснение кода:
  • Создаём Vue-приложение с состоянием role, которое может принимать значения 'admin', 'user' или 'guest'.
  • Используем v-if для отображения сообщения для администратора.
  • Используем v-else-if для отображения сообщения для обычного пользователя.
  • Используем v-else для отображения сообщения для гостя.

🚀 v-else-if — мощный инструмент для создания сложных условий отображения в Vue.js.

Практика: переключение текста

Этот текст можно скрыть!

<div id="toggle-demo">
  <button @click="toggleText">Переключить текст</button>
  <p v-if="isVisible">Этот текст можно скрыть!</p>
</div>

<script>
Vue.createApp({
  data() {
    return { isVisible: true }
  },
  methods: {
    toggleText() {
      this.isVisible = !this.isVisible;
    }
  }
}).mount("#toggle-demo");
</script>
Объяснение кода:
  • Создаём Vue-приложение с булевым состоянием isVisible.
  • Используем v-if для условного отображения текста.
  • Метод toggleText переключает значение isVisible при нажатии кнопки.

Реальные примеры циклов и условий во Vue

Рассмотрим несколько практических примеров использования директив v-for, v-if, v-else-if и v-show в реальных сценариях.

Пример 1. Список задач с фильтрацией

Фильтруем задачи по поисковому запросу:

  • {{ task.text }}

Нет задач!

Код:

<input type="text" v-model="search" placeholder="Поиск задачи">
<ul v-if="filteredTasks.length">
  <li v-for="task in filteredTasks" :key="task.id">{{ task.text }}</li>
</ul>
<p v-else>Нет задач!</p>

<script>
Vue.createApp({
  data() {
    return {
      search: "",
      tasks: [
        { id: 1, text: "Купить хлеб" },
        { id: 2, text: "Сделать уроки" },
        { id: 3, text: "Позвонить другу" }
      ]
    }
  },
  computed: {
    filteredTasks() {
      return this.tasks.filter(t => t.text.toLowerCase().includes(this.search.toLowerCase()));
    }
  }
}).mount("#tasks-demo");
</script>
Объяснение кода:
  • Используем v-model для двусторонней привязки поискового запроса.
  • Используем v-if для отображения списка задач, если есть совпадения.
  • Используем v-else для отображения сообщения, если нет задач.
  • Используем v-for для перебора отфильтрованных задач и их отображения.

Пример 2. Галерея с v-show

Показываем и скрываем галерею изображений по кнопке:

Код:

<button @click="isOpen = !isOpen">Показать/Скрыть галерею</button>
<div v-show="isOpen">
  <img v-for="img in images" :src="img" :key="img">
</div>
<script>
Vue.createApp({
  data() {
    return {
      isOpen: true,
      images: [
        "https://placehold.co/80x80?text=1",
        "https://placehold.co/80x80?text=2",
        "https://placehold.co/80x80?text=3"
      ]
    }
  }
}).mount("#gallery-demo");
</script>
Объяснение кода:
  • Используем v-show для управления видимостью галереи изображений.
  • Кнопка переключает значение isOpen, показывая или скрывая галерею.
  • Используем v-for для перебора массива изображений и их отображения.

Пример 3. Таблица пользователей с ролями

Отображаем роли пользователей с помощью v-if/v-else-if/v-else:

Имя Роль
{{ user.name }} 👑 Админ 👤 Пользователь 🚪 Гость

Код:

<tr v-for="user in users" :key="user.id">
  <td>{{ user.name }}</td>
  <td>
    <span v-if="user.role === 'admin'">👑 Админ</span>
    <span v-else-if="user.role === 'user'">👤 Пользователь</span>
    <span v-else>🚪 Гость</span>
  </td>
</tr>

<script>
Vue.createApp({
  data() {
    return {
      users: [
        { id: 1, name: "Алиса", role: "admin" },
        { id: 2, name: "Боб", role: "user" },
        { id: 3, name: "Гость", role: "guest" }
      ]
    }
  }
}).mount("#table-demo");
</script>
Объяснение кода:
  • Используем v-for для перебора массива пользователей и отображения их в таблице.
  • Используем v-if, v-else-if и v-else для отображения роли пользователя с соответствующей иконкой.
