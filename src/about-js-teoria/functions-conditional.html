Условия внутри функций

Условные операторы позволяют функции принимать решения и выполнять разные действия в зависимости от переданных данных.

Во время работы функции часто возникает необходимость проверить определенное условие. Например, определить, является ли число положительным, проверить возраст пользователя или убедиться, что введенные данные корректны. Для этого используются условные операторы.

Основные условные конструкции

  • if — выполняет блок кода, если условие истинно.
  • else — выполняет код, если условие ложно.
  • else if — позволяет проверить дополнительное условие.
  • switch — используется для выбора одного из нескольких вариантов выполнения кода.
  • && (И) — объединяет несколько условий, которые должны быть истинными одновременно.
  • || (ИЛИ) — достаточно, чтобы истинным было хотя бы одно условие.

Зачем использовать условия внутри функций?

  • Чтобы принимать решения во время выполнения функции.
  • Чтобы изменять поведение функции в зависимости от входных данных.
  • Чтобы выполнять разные действия при различных условиях.
  • Чтобы делать функции более универсальными и удобными для повторного использования.

Пример

function checkNumber(num) {
  if (num > 0) {
    return "Положительное число";
  } else if (num < 0) {
    return "Отрицательное число";
  } else {
    return "Ноль";
  }
}

В этом примере функция checkNumber() анализирует значение переменной num. Если число больше нуля, функция возвращает строку "Положительное число". Если число меньше нуля — "Отрицательное число". Во всех остальных случаях возвращается "Ноль".

Оператор if / else

Оператор if позволяет функции выполнять определенный блок кода только тогда, когда указанное условие истинно. Если условие не выполняется, можно использовать оператор else, чтобы выполнить другой блок кода.

Синтаксис

if (условие) {
  // код выполняется, если условие истинно
} else {
  // код выполняется, если условие ложно
}

Пример

function isEven(num) {
  if (num % 2 === 0) {
    return "Число четное";
  } else {
    return "Число нечетное";
  }
}

console.log(isEven(8));

В этом примере функция isEven() проверяет, делится ли число на 2 без остатка. Если условие истинно, функция возвращает строку "Число четное", иначе — "Число нечетное".

Использование else if

Если необходимо проверить несколько условий подряд, используется оператор else if. 

function checkNumber(num) {
  if (num > 0) {
    return "Положительное число";
  } else if (num < 0) {
    return "Отрицательное число";
  } else {
    return "Ноль";
  }
}

Сначала проверяется, больше ли число нуля. Если нет, проверяется второе условие. Если ни одно условие не выполнено, функция возвращает значение "Ноль".

Когда использовать if / else?

  • Когда нужно выполнить код только при выполнении определенного условия.
  • Когда необходимо выбрать один из нескольких вариантов выполнения программы.
  • Когда функция должна возвращать разные результаты в зависимости от входных данных.

💡 Не создавайте большое количество вложенных if, если можно упростить условие с помощью операторов &&, || или использовать switch. Это делает код более читаемым.

Операторы && и ||

Логические операторы && (И) и || (ИЛИ) используются внутри функций для проверки нескольких условий одновременно. Они помогают сделать код более компактным и понятным.

Оператор && (И)

Оператор && возвращает true только в том случае, если все условия истинны. 

function checkEligibility(age, subscription) {
  if (age >= 18 && subscription === "active") {
    return "Доступ разрешен";
  } else {
    return "Доступ запрещен";
  }
}

console.log(checkEligibility(20, "active"));

В данном примере функция проверяет сразу два условия: возраст пользователя должен быть не менее 18 лет, а подписка должна быть активной. Только если оба условия выполняются, функция разрешит доступ.

Оператор || (ИЛИ)

Оператор || возвращает true, если хотя бы одно из условий истинно. 

function canEnter(age, hasParent) {
  if (age >= 18 || hasParent) {
    return "Вход разрешен";
  } else {
    return "Вход запрещен";
  }
}

console.log(canEnter(15, true));

Здесь пользователь сможет войти, если ему уже исполнилось 18 лет или он находится с родителем.

Комбинирование операторов

Операторы && и || можно использовать вместе, чтобы создавать более сложные условия. 

function checkAccess(age, subscription, isAdmin) {
  if ((age >= 18 && subscription === "active") || isAdmin) {
    return "Доступ открыт";
  }

  return "Доступ закрыт";
}

В этом примере доступ получит пользователь, который имеет активную подписку и достиг 18 лет, или пользователь с правами администратора.

Когда использовать && и ||?

  • && — когда необходимо, чтобы выполнялись сразу несколько условий.
  • || — когда достаточно выполнения хотя бы одного условия.
  • Можно объединять оба оператора для создания более гибкой логики.

💡 Помните о приоритете операторов: сначала выполняется &&, затем ||. Если логика сложная, используйте круглые скобки — так код будет легче читать.
