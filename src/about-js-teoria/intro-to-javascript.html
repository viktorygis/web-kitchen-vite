Введение в JavaScript

JavaScript — это язык программирования, который был изначально создан для "оживления" веб-страниц.
Скрипты на JavaScript могут встраиваться в HTML и выполняться автоматически при загрузке страниц.

Основные факты о JavaScript:

  • Независимость от Java : Первоначально называемая "LiveScript", JavaScript была переименована в попытке воспользоваться популярностью Java. Сегодня это независимый язык со своей спецификацией ECMAScript.
  • Исполнение : JavaScript может выполняться в браузерах (с помощью различных движков, таких как V8 в Chrome и SpiderMonkey в Firefox) и на сервере (например, с использованием Node.js).
  • Безопасность : В браузере JavaScript ограничен в своих возможностях для защиты пользователей (например, не может читать/записывать произвольные файлы на жестком диске или взаимодействовать с другими вкладками).

Возможности JavaScript в браузере:

  • Изменение HTML-кода и стилей
  • Реагирование на действия пользователей
  • Выполнение сетевых запросов (AJAX)
  • Работа с cookies и локальным хранилищем

Ограничения JavaScript в браузере:

  • Не может выполнять операции с файловой системой без пользовательских действий
  • Не имеет доступа к другим вкладкам/доменам из-за политики "одинакового источника"

Преимущества JavaScript:

  1. Полная интеграция с HTML/CSS
  2. Простота использования для базовых задач
  3. Широкая поддержка всеми основными браузерами

Дополнительные языки и инструменты:
Существуют языки (например, TypeScript, CoffeeScript, Dart), которые транспилируются в JavaScript, предлагая дополнительные возможности, такие как строгая типизация или более лаконичный синтаксис.

Спецификация JavaScript

ECMA-262 — это официальная спецификация, содержащая самую глубокую, детальную и формализованную информацию о JavaScript.
Она определяет сам язык.

Вначале спецификация может показаться тяжеловатой для понимания из-за слишком формального стиля изложения.
Если вы ищете источник самой достоверной информации, то это правильное место, но она не для ежедневного использования.

Где найти спецификацию?

  • Текущая версия : Новая версия спецификации появляется каждый год
  • Черновик : tc39.es/ecma262
  • Предложения : GitHub proposals (stage 3 proposals)

Особенности:

  • Формальное описание языка
  • Ежегодные обновления
  • Содержит все новые возможности

Справочники по JavaScript

MDN (Mozilla Developer Network) — это наиболее полный и авторитетный справочник по JavaScript с примерами и другой информацией.

MDN JavaScript Reference

Хороший источник для получения подробных сведений о функциях языка, методах встроенных объектов и так далее.

Располагается по адресу developer.mozilla.org .

Как искать информацию?

Часто удобнее использовать поисковик с запросом «MDN [что вы хотите найти]», например поиск информации о функции parseInt .

Таблицы совместимости JavaScript

JavaScript — это развивающийся язык, в который постоянно добавляется что-то новое.
Таблицы совместимости помогают понять, какие возможности поддерживаются в разных браузерах.

caniuse.com

caniuse.com — таблицы с информацией о поддержке по каждой возможности языка.

Пример: поддержка криптографии .

kangax.github.io

kangax.github.io/compat-table — таблица с возможностями языка и движками, которые их поддерживают.

Эти ресурсы полезны в ежедневной работе программиста, так как содержат ценную информацию о возможностях использования языка и их поддержке.
