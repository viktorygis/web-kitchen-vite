Циклы внутри функций

Циклы позволяют функции многократно выполнять один и тот же блок кода без его повторного написания.

Во время работы функции часто возникает необходимость обработать несколько элементов массива, выполнить действие определенное количество раз или повторять операции до выполнения заданного условия. Для этого используются циклы.

Основные виды циклов

  • for — выполняет код заданное количество раз.
  • while — выполняет код, пока условие остается истинным.
  • do...while — сначала выполняет код, затем проверяет условие.

Зачем использовать циклы внутри функций?

  • Чтобы не писать одинаковый код несколько раз.
  • Чтобы обрабатывать массивы и коллекции данных.
  • Чтобы выполнять действия нужное количество раз.
  • Чтобы сделать функции более компактными и удобными.

Пример

function printNumbers() {
  for (let i = 1; i <= 5; i++) {
    console.log(i);
  }
}

printNumbers();

В этом примере функция с помощью цикла for выводит числа от 1 до 5. Вместо пяти одинаковых команд используется один цикл.

Цикл for

Цикл for используется тогда, когда заранее известно, сколько раз необходимо выполнить определенный блок кода.

Синтаксис

for (начало; условие; изменение) {
  // выполняемый код
}

Пример

function printNumbers() {
  for (let i = 1; i <= 5; i++) {
    console.log(i);
  }
}

printNumbers();

Сначала создается переменная i, затем перед каждой итерацией проверяется условие i <= 5. После выполнения каждой итерации значение i увеличивается на единицу.

Когда использовать for?

  • Когда известно количество повторений.
  • При переборе элементов массива.
  • Для выполнения однотипных действий несколько раз.

💡 Если известно количество повторений, чаще всего используют именно цикл for.

Цикл while

Цикл while выполняет код до тех пор, пока указанное условие остается истинным. Количество повторений заранее может быть неизвестно.

Синтаксис

while (условие) {
  // выполняемый код
}

Пример

function countdown() {
  let count = 5;

  while (count > 0) {
    console.log(count);
    count--;
  }

  console.log("Старт!");
}

countdown();

Функция выводит числа от 5 до 1. После каждой итерации значение переменной count уменьшается на единицу. Когда условие становится ложным, цикл завершается.

Когда использовать while?

  • Когда неизвестно количество повторений.
  • Когда выполнение зависит от определенного условия.
  • Когда необходимо выполнять действия до получения нужного результата.

💡 Не забывайте изменять переменную внутри цикла while. Иначе цикл может стать бесконечным.

Цикл do...while

Цикл do...while выполняет блок кода один раз, а затем проверяет условие. Если условие истинно, цикл повторяется. Отличие от обычного while в том, что код выполняется минимум один раз.

Синтаксис

do {
  // выполняемый код
} while (условие);

Пример

function askNumber() {
  let input;

  do {
    input = prompt("Введите число больше 10:");
  } while (input <= 10);

  console.log("Спасибо! Вы ввели: " + input);
}

askNumber();

В этом примере функция запрашивает у пользователя число. Даже если пользователь введет неправильное число с первой попытки, цикл do...while выполнится хотя бы один раз, а затем продолжит запрашивать число, пока оно не будет больше 10.

Когда использовать do...while?

  • Когда код должен выполниться хотя бы один раз.
  • При запросе ввода у пользователя с валидацией.
  • Когда нужно повторить действие до выполнения условия.

💡 do...while полезен при работе с пользовательским вводом, когда нужно гарантировать, что действие произойдет минимум один раз.

Сравнение циклов for и while

И for, и while позволяют многократно выполнять один и тот же блок кода. Однако они используются в разных ситуациях.

Когда использовать for?

  • Когда известно количество повторений.
  • При переборе массива.
  • Когда нужен индекс элемента.

Когда использовать while?

  • Когда заранее неизвестно количество повторений.
  • Когда цикл зависит от выполнения некоторого условия.
  • Когда управление циклом должно быть более гибким.

Для перебора массивов в JavaScript чаще используют for, потому что размер массива известен через свойство length. Цикл while подходит в тех случаях, когда завершение работы зависит не только от количества элементов, но и от других условий.

💡 Для перебора массива предпочтительно использовать for. Цикл while чаще применяется для решения задач, где количество повторений заранее неизвестно.

Циклы и перебор массивов

Перебор массива — это одно из самых частых применений циклов. Циклы позволяют последовательно получить доступ к каждому элементу массива и выполнить с ним необходимые действия.

Перебор с помощью цикла for

Цикл for идеально подходит для перебора массивов, так как мы можем использовать индекс для доступа к элементам. 

function printArray(arr) {
  for (let i = 0; i < arr.length; i++) {
    console.log(arr[i]);
  }
}

printArray(['Яблоко', 'Банан', 'Апельсин']);

Функция перебирает массив от начала до конца. На каждой итерации переменная i увеличивается на 1, и мы получаем доступ к элементу под индексом i. Цикл продолжается, пока i меньше длины массива.

Перебор с помощью цикла while

Цикл while также можно использовать для перебора, но это менее удобно, чем for. 

function printArray(arr) {
  let i = 0;

  while (i < arr.length) {
    console.log(arr[i]);
    i++;
  }
}

printArray(['Яблоко', 'Банан', 'Апельсин']);

Здесь мы вручную инициализируем счетчик i, проверяем условие и увеличиваем счетчик. Это работает, но требует больше кода.

Вычисление суммы элементов массива

function sumArray(arr) {
  let sum = 0;

  for (let i = 0; i < arr.length; i++) {
    sum += arr[i];
  }

  return sum;
}

console.log(sumArray([5, 10, 15, 20])); // 50

Функция проходит по каждому элементу массива и добавляет его к переменной sum. После завершения цикла функция возвращает сумму.

Поиск элемента в массиве

function findElement(arr, target) {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] === target) {
      return i;
    }
  }

  return -1; // элемент не найден
}

console.log(findElement([10, 20, 30, 40], 30)); // 2

Функция ищет первое совпадение элемента в массиве. Если элемент найден, возвращается его индекс. Если элемент не найден, возвращается -1.

Подсчет элементов с условием

function countEven(arr) {
  let count = 0;

  for (let i = 0; i < arr.length; i++) {
    if (arr[i] % 2 === 0) {
      count++;
    }
  }

  return count;
}

console.log(countEven([1, 2, 3, 4, 5, 6])); // 3

Функция подсчитывает количество четных чисел в массиве. Каждый элемент проверяется на четность с помощью оператора модуля.

Типичные операции при переборе массивов

  • Вычисление суммы или среднего — аккумулирование значений.
  • Поиск элемента — нахождение элемента по условию.
  • Подсчет элементов — определение количества элементов, соответствующих условию.
  • Преобразование элементов — изменение каждого элемента.
  • Создание нового массива — выбор или преобразование элементов из исходного.

💡 Цикл for — самый удобный способ перебора массивов в JavaScript. Используйте arr.length для определения количества элементов.

Фильтрация элементов массива

Во время перебора массива можно выбирать только те элементы, которые удовлетворяют определенному условию. Такой процесс называется фильтрацией.

Пример

function filterGreaterThanFive(arr) {
  let greaterThanFive = [];

  for (let i = 0; i < arr.length; i++) {
    if (arr[i] > 5) {
      greaterThanFive.push(arr[i]);
    }
  }

  console.log(greaterThanFive);
}

filterGreaterThanFive([1, 3, 7, 10, 2]);

Цикл последовательно проходит по каждому элементу массива. Если элемент больше 5, он добавляется в новый массив с помощью метода push().

После завершения цикла новый массив будет содержать только подходящие элементы.

Фильтрация по строке

function filterByLength(arr, length) {
  let result = [];

  for (let i = 0; i < arr.length; i++) {
    if (arr[i].length === length) {
      result.push(arr[i]);
    }
  }

  return result;
}

console.log(filterByLength(['кот', 'слон', 'пес', 'собака'], 3));

Функция отбирает только строки, длина которых равна заданному значению.

Когда используется фильтрация?

  • Поиск чисел больше или меньше заданного значения.
  • Выбор пользователей определенного возраста.
  • Отбор товаров по цене.
  • Получение только нужных элементов массива.
