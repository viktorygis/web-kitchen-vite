Массивы в JavaScript

Массив — это упорядоченная коллекция данных, которая может содержать элементы разных типов.
В JavaScript массивы являются объектами и имеют специальные методы для работы с элементами.

Основные характеристики:

  • Индексация начинается с 0
  • Могут изменяться (мутировать)
  • Автоматически изменяют длину
  • Могут содержать любые типы данных

Пример массива:

const mixedArray = [
  'текст',
  42,
  true,
  { name: 'Объект' },
  [1, 2, 3]
];

Синтаксис:

Если обычная переменная хранит одно значение, то массив позволяет хранить несколько в квадратных скобках [ ], разделяя элементы запятыми.

let array = [element1, element2, element3];

Примеры:

let fruits = ["яблоко", "банан", "вишня"];
let numbers = [1, 2, 3, 4];

Может хранить разные типы данных:

Массивы могут содержать элементы разных типов, включая числа, строки, объекты и даже другие массивы. 

let mixed = [1, "текст", true, null, undefined, { name: "Иван" }];
let simple = ["hello", 25];

Создание пустого массива:

Пустой массив можно создать с помощью литерала [] или конструктора new Array().
Потом можно добавлять элементы с помощью методов массива, таких как push(). 

let guestList = []; // Пустой массив
guestList.push("Леха", "Лепеха"); // Добавление элементов
console.log(guestList); // ["Леха", "Лепеха"]

Переприсвоение массива:

Если переменная объявлена с помощью let, ей можно присвоить совершенно новый массив целиком. 

let foods = ["овощи", "фрукты"];
foods = ["чипсики", "энергетики"]; // Переменной присвоен новый массив
console.log(foods); // Выведет ["чипсики", "энергетики"]

Индексы:

Индексы — это числовые позиции элементов в массиве, начиная с нуля. Первый элемент имеет индекс 0, второй — 1 и так далее.
Для доступа к элементу массива используйте квадратные скобки с индексом. 

let fruits = ["яблоко", "банан", "вишня"];
console.log(fruits[0]); // "яблоко"
console.log(fruits[1]); // "банан"
console.log(fruits[2]); // "вишня"

Обращение к несуществующему элементу:

Если обратиться к индексу, который не существует, вернется undefined.

let items = ["ручка", "ластик", "тетрадь"];

console.log(items[1]); // "ластик" ✅ (существует)
console.log(items[5]); // undefined ❌ (такого индекса нет)

Массив через const:

Даже если массив объявлен с помощью const, его элементы можно изменять, добавлять и удалять. Однако нельзя переназначить саму переменную массива.

Assignment to constant variable. — ошибка, возникающая при попытке переназначить переменную, объявленную с помощью const.

const numbers = [1, 2, 3];
numbers[1] = 42; // Изменяем второй элемент
console.log(numbers); // [1, 42, 3]

// Попытка переназначить массив вызовет ошибку
// numbers = [4, 5, 6]; // ❌ Ошибка: Assignment to constant variable.

Замена элемента по индексу:

Чтобы заменить любой элемент массива, нужно просто указать его индекс и новое значение. 

let fruits = ["яблоко", "банан", "арбузик"];
fruits[1] = "клубничка"; // Меняем "банан" на "клубничку"
console.log(fruits); // ["яблоко", "клубничка", "арбузик"]

Свойство .length

Свойство .length возвращает количество элементов в массиве. 

let fruits = ["яблоко", "банан", "арбузик"];
console.log(fruits.length); // 3

length - 1

Чтобы получить индекс последнего элемента массива, нужно использовать length - 1. 

let fruits = ["яблоко", "банан", "арбузик"];
console.log(fruits[fruits.length - 1]); // "арбузик"

// Изменяем последний элемент массива
let numbers = [213, 44, 95, 23, 71, 85, 12];
numbers[numbers.length - 1] = 3;
console.log(numbers);

Проверка на пустой массив

Чтобы проверить, пустой ли массив, можно использовать свойство .length. Если длина равна 0, массив пуст. 

let emptyArray = [];
if (emptyArray.length === 0) {
  console.log("Массив пустой");
} else {
  console.log("Массив содержит элементы");
}

Удаление всех элементов массива

Чтобы удалить все элементы массива, можно присвоить ему пустой массив. 

let fruits = ["яблоко", "банан", "арбузик"];
fruits = []; // Теперь массив пустой
console.log(fruits); // []

Полная замена массива

Чтобы заменить весь массив, можно присвоить переменной новый массив. 

let fruits = ["яблоко", "банан", "арбузик"];
fruits = ["груша", "апельсин"]; // Теперь массив содержит новые элементы
console.log(fruits); // ["груша", "апельсин"]

Создание массивов

В JavaScript есть несколько способов создания массивов, каждый из которых имеет свои особенности.

Способ Пример Примечание
Литерал массива const arr = [1, 2, 3]; Наиболее распространенный способ
Конструктор Array const arr = new Array(1, 2, 3); Не рекомендуется для простых случаев
Array.of() const arr = Array.of(1, 2, 3); Полезен для единичных числовых элементов
Array.from() const arr = Array.from('123'); Создает массив из итерируемого или array-like объекта
Пустой массив + заполнение const arr = []; arr[0] = 1; Постепенное добавление элементов

Особенности создания:

  • Массивы, созданные литералом, конструктором с явными элементами, Array.of() или Array.from(), дают одинаковый результат — обычный массив с заполненными элементами
  • Конструктор new Array(n) с единственным числовым аргументом — особый случай: он создает массив длины n с пустыми слотами (empty slots), а не с элементами undefined. Такой массив ведет себя иначе в методах forEach, map, JSON.stringify и др.
  • Array.from() может принимать функцию преобразования вторым аргументом

Основные методы массивов

JavaScript предоставляет множество полезных методов для работы с массивами.
Методы можно разделить на мутирующие (изменяют исходный массив) и неммутирующие.

Мутирующие методы:

  • push() - добавляет элемент в конец
  • pop() - удаляет последний элемент
  • shift() - удаляет первый элемент
  • unshift() - добавляет элемент в начало
  • splice() - добавляет/удаляет элементы
  • sort() - сортирует массив
  • reverse() - переворачивает массив

Немутирующие методы:

  • concat() - объединяет массивы
  • slice() - возвращает часть массива
  • join() - объединяет элементы в строку
  • indexOf() - ищет индекс элемента
  • includes() - проверяет наличие элемента
  • find() - ищет элемент по условию
  • filter() - фильтрует элементы
  • map() - преобразует элементы

Примеры использования методов:

const numbers = [1, 2, 3];

// Добавление элементов
numbers.push(4); // [1, 2, 3, 4]
numbers.unshift(0); // [0, 1, 2, 3, 4]

// Удаление элементов
numbers.pop(); // [0, 1, 2, 3]
numbers.shift(); // [1, 2, 3]

// Поиск и фильтрация
const even = numbers.filter(n => n % 2 === 0); // [2]
const doubled = numbers.map(n => n * 2); // [2, 4, 6]

Распространенные операции с массивами

Массивы используются для решения множества задач. Рассмотрим наиболее частые операции.

1. Итерация по массиву

const fruits = ['apple', 'banana', 'orange'];

// forEach
fruits.forEach((fruit, index) => {
  console.log(`${index}: ${fruit}`);
});

// for..of
for (const fruit of fruits) {
  console.log(fruit);
}

2. Копирование массива

const original = [1, 2, 3];

// Способ 1: spread оператор
const copy1 = [...original];

// Способ 2: slice()
const copy2 = original.slice();

// Способ 3: Array.from()
const copy3 = Array.from(original);

3. Объединение массивов

const arr1 = [1, 2];
const arr2 = [3, 4];

// Способ 1: concat()
const combined1 = arr1.concat(arr2);

// Способ 2: spread оператор
const combined2 = [...arr1, ...arr2];

4. Проверка массива

const arr = [1, 2, 3];

// Проверка, что это массив
console.log(Array.isArray(arr)); // true

// Проверка наличия элемента
console.log(arr.includes(2)); // true
console.log(arr.indexOf(3) !== -1); // true

// Проверка всех элементов
const allPositive = arr.every(n => n > 0); // true

Продвинутые операции:

Деструктуризация

const rgb = [255, 128, 0];

// Извлечение значений
const [red, green, blue] = rgb;
console.log(red); // 255

Редукция (reduce)

const nums = [1, 2, 3, 4];

// Сумма элементов
const sum = nums.reduce((acc, n) => acc + n, 0);
console.log(sum); // 10

Рекомендации по работе с массивами:

  • Используйте методы массивов вместо циклов, где это возможно
  • Для сложных операций комбинируйте методы (map + filter + reduce)
  • Избегайте мутаций исходных массивов, если это не требуется
  • Для больших массивов используйте typed arrays (Int32Array и др.)
