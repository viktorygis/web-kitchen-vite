Базовые операторы в JavaScript

Операторы — символы, выполняющие операции над значениями.
В JavaScript операторы делятся на несколько категорий по количеству операндов.

Типы операторов:

  • Унарные — один операнд ( -x , ++x )
  • Бинарные — два операнда ( x + y )
  • Тернарные — три операнда ( a ? b : c )

Основные категории:

  • Арифметические
  • Присваивания
  • Сравнения
  • Логические

Математические операторы

Арифметика — базовые математические операции с числами.
JavaScript поддерживает все основные операции.

Оператор Название Пример Результат
+ Сложение 3 + 2
- Вычитание 5 - 2
* Умножение 2 * 3
/ Деление 6 / 2
% Остаток 5 % 2
** Степень 2 ** 3
+ Конкатенация 'a' + 'b'

Особенности:

  • Деление / всегда возвращает число (даже целочисленное)
  • Остаток % сохраняет знак делимого
  • Оператор + преобразует числа в строки при работе со строками
  • Остаток от деления полезен для проверки делимости чисел (например, чётное/нечётное)

Приоритет операций:

  1. () (скобки)
  2. ** (степень)
  3. * , / , %
  4. + , -

Пример с приоритетами:

let result = 5 + 2 * 3; // 11 (умножение выполняется первым)
let result = (5 + 2) * 3; // 21 (скобки меняют порядок)

Примеры использования:

// Сложение
let a = 10;
let b = 40;
let sum = a + b;
console.log(`Сложение чисел: ${sum}`); // 50

// Вычитание
let f = b - a;
console.log(`Вычитание чисел: ${f}`); // 30

// Умножение
let y = a * b;
console.log(`Умножение чисел: ${y}`); // 400

// Деление
let d = b / a;
console.log(`Деление чисел: ${d}`); // 4

// Остаток от деления
let remainder = b % a;
console.log(`Остаток от деления(40/10): ${remainder}`); // 0

let remainder02 = 40 % 7;
console.log(`Остаток от деления(40/7): ${remainder02}`); // 5

Операторы присваивания

Присваивание — сохранение значения в переменной.
В JavaScript есть базовые и составные операторы присваивания.

Оператор Пример Эквивалент Демо
= x = 5 -
+= x += 2 x = x + 2
-= x -= 1 x = x - 1
*= x *= 3 x = x * 3
/= x /= 2 x = x / 2

Особенности:

  • Присваивание возвращает значение ( a = b = c = 5 )
  • Составные операторы выполняют операцию и присваивание
  • Можно использовать с любыми бинарными операторами

Цепочка присваиваний:

let a, b, c;
a = b = c = 5; // все переменные получат значение 5

Другие операторы

Инкремент/декремент

Изменяют значение переменной на 1:

Форма Оператор Пример Демо
Префиксная ++x , --x let x = 1; alert(++x); // 2
Постфиксная x++ , x-- let x = 1; alert(x++); // 1

Оператор запятая

Вычисляет несколько выражений, возвращая последнее:

let x = (1 + 1, 2 + 2); // x = 4

Практические советы:

  • Используйте скобки для явного указания порядка операций
  • Избегайте сложных выражений с побочными эффектами
  • Будьте осторожны с инкрементом в составных выражениях
  • Используйте составные операторы для краткости кода
