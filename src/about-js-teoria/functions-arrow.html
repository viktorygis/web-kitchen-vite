Стрелочные функции в JavaScript

Стрелочные функции — это короткий и удобный синтаксис для записи функций в JavaScript. Они особенно хорошо подходят для небольших вычислений, колбэков и простых обработчиков. У стрелочных функций нет собственного this, поэтому они берут его из внешнего контекста.

Стрелочная функция записывается с помощью стрелки =>.

Простой пример

const greet = () => {
  console.log("Привет, мир!");
};

greet(); // Привет, мир!

То же самое обычной функцией

function greet() {
  console.log("Привет, мир!");
}

greet(); // Привет, мир!

Объявление

Стрелочная функция — это разновидность функционального выражения. Если тело состоит из одного выражения, фигурные скобки можно опустить, а результат вернётся автоматически. 

const sum = (a, b) => a + b;
console.log(sum(1, 2)); // 3

Если нужны несколько строк, используйте фигурные скобки и явно пишите return. 

const sum = (a, b) => {
  return a + b;
};

console.log(sum(1, 2)); // 3

Параметры и аргументы

Параметры — это имена переменных в объявлении функции. Аргументы — это значения, которые передаются при вызове.

Один параметр

Если параметр один, круглые скобки можно опустить.

const square = n => n * n;
console.log(square(4)); // 16

Несколько параметров

Если параметров несколько, скобки обязательны.

const multiply = (a, b) => a * b;
console.log(multiply(3, 4)); // 12

Без параметров

Если параметров нет, скобки всё равно нужны.

const sayHi = () => alert("Hello!");
sayHi();

Пример с именем

const greetName = name => {
  console.log("Привет, " + name + "!");
};

greetName("Иван"); // Привет, Иван!

Return

Если стрелочная функция записана без фигурных скобок, она возвращает результат выражения автоматически. Если используются фигурные скобки, нужно явно писать return.

Неявный возврат

const sum = (a, b) => a + b;
console.log(sum(2, 3)); // 5

Явный возврат

const calculate = (a, b) => {
  const result = a + b;
  return result * 2;
};

console.log(calculate(2, 3)); // 10

Если забыть return внутри блока, функция вернёт undefined.

Контекст this

У стрелочной функции нет собственного this. Она берёт его из внешней области видимости, поэтому хорошо подходит для вложенных колбэков, но не для методов объекта.

Стрелочная функция внутри метода

const person = {
  name: "Иван",
  greet() {
    const inner = () => {
      console.log(this.name);
    };

    inner();
  }
};

person.greet(); // Иван

Стрелочная функция как метод

const person = {
  name: "Иван",
  greet: () => {
    console.log(this.name);
  }
};

person.greet(); // undefined

В обычном методе this указывает на объект, а у стрелочной функции значение this берётся снаружи.

Динамическое создание

Стрелочные функции можно создавать на лету, если выбор зависит от условия. 

const age = prompt("Сколько вам лет?", 18);

const welcome = age < 18
  ? () => alert("Привет!")
  : () => alert("Здравствуйте!");

welcome();

Многострочные функции

Если нужно выполнить несколько действий, используйте фигурные скобки. В этом случае return пишется явно. 

const formatSum = (a, b) => {
  const result = a + b;
  return "Сумма: " + result;
};

console.log(formatSum(2, 3)); // Сумма: 5

Это удобно, когда функция делает не одно действие, а небольшой набор шагов.

Итого

  • Стрелочные функции — это короткий и удобный синтаксис.
  • Если тело состоит из одного выражения, результат возвращается автоматически.
  • Если используются фигурные скобки, нужен явный return.
  • Один параметр можно писать без скобок.
  • У стрелочных функций нет собственного this.
  • Они особенно удобны для колбэков, коротких обработчиков и простых вычислений.
