Математические операторы и округление в JavaScript

В этом разделе — основные арифметические операторы, способы округления, работа с числами и парсинг строк, генерация случайных чисел и подводные камни точности с числами с плавающей точкой.

Операторы:

  • Арифметика: + - * / % **
  • Унарные: +x (приведение к числу), -x (смена знака)
  • Скобки: ( ) для порядка операций

Округление:

  • Math.floor , Math.ceil , Math.round , Math.trunc
  • toFixed для форматирования с количеством знаков

Арифметические операторы

JavaScript поддерживает все базовые операции. Деление всегда возвращает число с плавающей точкой (если нужно).

Оператор Название Пример Код (JS) Демо
+ Сложение 3 + 2 
console.log(3 + 2); // 5
- Вычитание 5 - 2 
console.log(5 - 2); // 3
* Умножение 2 * 3 
console.log(2 * 3); // 6
/ Деление 7 / 2 
console.log(7 / 2); // 3.5
% Остаток 7 % 2 
console.log(7 % 2); // 1
** Степень 2 ** 5 
console.log(2 ** 5); // 32
+x Унарный плюс +'42' 
const n = +'42';
console.log(n, typeof n); // 42 'number'
-x Смена знака -10 
let x = 10;
console.log(-x); // -10

Особенности:

  • + со строкой выполняет конкатенацию: '1' + 2 = '12'
  • Унарный + приводит к числу: +'1.5' = 1.5
  • % — остаток, знак совпадает со знаком делимого
  • Приоритет: () → ** → * / % → + -

Демо приоритета:

let a = 5 + 2 * 3;      // 11
let b = (5 + 2) * 3;    // 21

Округление чисел

Четыре основных функции округления из Math , и форматирование через toFixed .

Функция Описание Пример Демо
Math.floor(x) Вниз до целого Math.floor(3.9) = 3
Math.ceil(x) Вверх до целого Math.ceil(3.1) = 4
Math.round(x) До ближайшего Math.round(3.5) = 4
Math.trunc(x) Отбрасывает дробную часть Math.trunc(-3.9) = -3
num.toFixed(n) Форматирует с n знаками после точки (строка) (1.005).toFixed(2)

Советы:

  • Для округления до n знаков: Math.round(x * 10**n) / 10**n
  • toFixed возвращает строку — для числа используйте Number(x.toFixed(n))
  • Поведение Math.round(0.5) → 1; Math.round(-0.5) → 0 (симметрия относительно 0 не строгая)

Парсинг строк и проверки чисел

Парсинг

Number('42')      // 42
parseInt('101',2) // 5 (база 2)
parseFloat('3.14px') // 3.14

Проверки

Number.isNaN(value)     // строго NaN
Number.isFinite(value)  // конечное число
isNaN('foo')            // true (устаревшая нестрогая)

Рекомендуется: Number.isNaN и Number.isFinite — они не приводят тип.

Разделители разрядов и основы

  • Числовые разделители: 1_000_000 (для читаемости)
  • Разные системы счисления: 0b1010 (2), 0o77 (8), 0xFF (16)

Случайные числа

Math.random() возвращает число в диапазоне [0, 1) . На его основе можно строить нужные диапазоны.

Формулы:

// [min, max) вещественное
min + Math.random() * (max - min)

// [min, max] целое
Math.floor(min + Math.random() * (max - min + 1))

Точность и ошибки округления

Двоичное представление ломает «идеальные» десятичные дроби: 0.1 + 0.2 !== 0.3 . Используйте округление или сравнение с учетом Number.EPSILON . 

// Проблема
0.1 + 0.2 === 0.3 // false

// Решения
// 1) Округлить до n знаков
Math.round((0.1 + 0.2) * 100) / 100 // 0.3

// 2) Почти равны с EPSILON
const nearlyEqual = (a, b, eps = Number.EPSILON) => Math.abs(a - b) < eps;

Полезные математические функции

Частые методы Math

Math.abs(-5)     // 5
Math.max(1,4,2) // 4
Math.min(1,4,2) // 1
Math.pow(3,4)   // 81
Math.sqrt(16)   // 4
Math.hypot(3,4) // 5 (гипотенуза)
Math.sign(-10)  // -1

Зажатие значения (clamp)

const clamp = (x, min, max) => Math.min(Math.max(x, min), max);
clamp(120, 0, 100) // 100
