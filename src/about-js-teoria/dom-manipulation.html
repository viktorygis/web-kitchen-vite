Манипуляция с элементами DOM

Манипуляция DOM — основа для создания интерактивных сайтов. С помощью JavaScript ты можешь добавлять уведомления, товары, комментарии и многое другое прямо на страницу!

Что можно делать:

Изменять текст и HTML

Меняйте содержимое элемента: element.textContent — для текста,
element.innerHTML — для HTML-разметки.

Менять стили

Управляйте инлайн-стилями элемента, например: element.style.color = 'blue' .

Работать с классами

Добавляйте, удаляйте и переключайте классы через element.classList.add() ,
.remove() ,
.toggle() .

Создавать и удалять элементы

Создавайте узлы и вставляйте в DOM: document.createElement() ,
удаляйте через element.remove()

Изменять текст и HTML

У элементов есть два базовых способа работы с содержимым: textContent и innerHTML .

textContent

textContent — устанавливает или возвращает чистый текст.
Любые теги будут отображены как текст. Это безопаснее и обычно быстрее.

Примеры:

Пример: заменить текст (textContent)

// HTML:
		// <h1 id="greeting">Привет, мир!</h1>

		const greeting = document.getElementById('greeting');
		greeting.textContent = 'Добро пожаловать!';
		console.log(greeting.textContent); // "Добро пожаловать!"

Получить текущее содержимое (textContent)

// HTML:
	// <p id="text">Это текст</p>

	const textElement = document.getElementById('text');
	console.log(textElement.textContent); // "Это текст"

Пример: очистить содержимое (textContent)

// HTML:
	// <div id="content">Привет, это содержимое!</div>

	const content = document.getElementById('content');
	content.textContent = ''; // безопасно для любого содержимого

innerHTML

innerHTML — устанавливает или возвращает HTML-разметку.
Строка парсится как HTML, теги превращаются в элементы.

Примеры:

вставить HTML (innerHTML)

// HTML:
		// <div id="container">Здесь что-то важное</div>

		const container = document.getElementById('container');
		container.innerHTML = '<b>Важный текст!</b>';
		console.log(container.innerHTML); // "<b>Важный текст</b>"

Получить текущее содержимое (innerHTML)

// HTML:
		// <p><b>Жирный текст</b></p>
		const textElement = document.getElementById('text');
		console.log(textElement.innerHTML); // "<b>Жирный текст</b>"
		console.log(textElement.textContent); // "Жирный текст"

Очистить содержимое (innerHTML)

// HTML:
		// <div id="content">Привет, это содержимое!</div>

		const content = document.getElementById('content');
		content.innerHTML = '';   // удалит всю разметку внутри

Когда что использовать?

  • Если нужно просто поменять текст — используй textContent (быстрее и безопаснее).
  • Если нужно добавить разметку — используй innerHTML или создавай узлы через createElement и append .

Безопасность

  • Никогда не передавай в innerHTML непроверенные данные пользователей — это риск XSS.
  • Если данные приходят извне, используй textContent , специальные санитайзеры, либо создавай элементы вручную: 
const msg = document.createElement('b');
msg.textContent = userInput; // безопасно — теги экранируются
container.replaceChildren(msg);

Создавать и удалять элементы

В работе с DOM есть два ключевых процесса: создание новых элементов и удаление уже существующих.
Оба действия делаются через специальные методы объекта document и самих элементов.

Создать элемент

createElement — создает новый DOM-узел (но НЕ вставляет его в страницу).

Примеры:

Пример: создать и добавить новый элемент

// Создать элемент:
const newElement = document.createElement('p');
// Наполнить содержимым:
newElement.textContent = 'Привет, я новый элемент!';
// Вставить в DOM:
const container = document.getElementById('container');
container.appendChild(newElement);

Современный способ: append

// Можно добавить сразу текст или элементы:
container.append('Привет, я текст!');
container.append(newElement);

Перемещение элемента

// Если элемент уже есть в DOM, он будет перемещён:
document.body.appendChild(newElement); // newElement переедет в body

Удалять элементы

removeChild — удаляет дочерний элемент через родителя.
remove — удаляет элемент напрямую.

Примеры:

Удалить через родителя (removeChild)

const container = document.getElementById('container');
const text = document.getElementById('text');
container.removeChild(text);

Современный способ: remove

const text = document.getElementById('text');
text.remove();

Когда что использовать?

  • Чтобы добавить новый элемент — createElement + appendChild / append .
  • Чтобы удалить — remove (проще и быстрее), либо removeChild , если нужен контроль через родителя.
  • Если элемент добавляется повторно — он перемещается, а не клонируется.

Безопасность

  • Никогда не вставляй в DOM элементы с несанитизированным innerHTML .
  • Для динамического создания — всегда используй createElement и textContent , чтобы избежать XSS. 
const msg = document.createElement('b');
msg.textContent = userInput; // безопасно, даже если userInput содержит теги!
container.replaceChildren(msg);

Динамическое создание множества элементов

Для динамического добавления сразу нескольких элементов (например, пунктов списка, карточек, блоков) — используй циклы и createElement .
Это универсальный способ для работы с массивами данных и шаблонами.

Добавить сразу несколько элементов

createElement — создает новый элемент; appendChild (или append ) — вставляет в DOM.
Всё повторяется в цикле!

Примеры:

Пример: добавить 3 пункта списка

// HTML:
<ul id="my-list"></ul>

// JS:
const list = document.getElementById("my-list");
for (let i = 1; i <= 3; i++) {
  const listItem = document.createElement("li");
  listItem.textContent = `Пункт ${i}`;
  list.appendChild(listItem);
}

Добавление карточек из массива

const users = ["Иван", "Ольга", "Сергей"];
const box = document.getElementById("cards");
users.forEach(name => {
  const card = document.createElement("div");
  card.className = "user-card";
  card.textContent = name;
  box.appendChild(card);
});

Оптимизация: вставка одним действием

// Сначала в память, потом в DOM:
const frag = document.createDocumentFragment();
for (let i = 1; i <= 5; i++) {
  const li = document.createElement("li");
  li.textContent = `Пункт ${i}`;
  frag.appendChild(li);
}
list.appendChild(frag);

Удалять сразу несколько элементов

removeChild — удаляет по одному; replaceChildren([]) — современный способ очистить контейнер.
Всё делается в цикле или разом!

Примеры:

Удалить все элементы списка вручную

const list = document.getElementById("my-list");
while (list.firstChild) {
  list.removeChild(list.firstChild);
}

Современный способ: replaceChildren

list.replaceChildren(); // Мгновенно удаляет все дочерние элементы

Когда что использовать?

  • Для генерации большого количества элементов — всегда используйте createElement в цикле.
  • Для очистки контейнера — используйте replaceChildren или удаляйте по одному в цикле.
  • Для ускорения — используйте DocumentFragment и вставку одним действием.

Безопасность

  • В цикле всегда используйте textContent для добавления текста.
  • Не вставляйте массивы данных через innerHTML — это опасно для XSS! 
const li = document.createElement('li');
li.textContent = userInput; // безопасно, даже если userInput содержит теги!
list.appendChild(li);
