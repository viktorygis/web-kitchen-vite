Введение Инструменты Стек PM FAQ

Инструменты Product Manager

Инструменты Product Manager — это программы и сервисы, которые помогают исследовать пользователей, управлять задачами, анализировать данные и принимать продуктовые решения.

Какие задачи решают инструменты PM

  • Сбор информации о пользователях.
  • Документирование продуктовых решений.
  • Управление backlog и roadmap.
  • Анализ поведения пользователей.
  • Мониторинг продуктовых метрик.
  • Коммуникация с командой.

Основные инструменты Product Manager

Product Manager использует разные инструменты на разных этапах создания и развития продукта.

Категория Инструменты Для чего используются
Управление задачами Jira, Linear, Asana Backlog, sprint, epic, user story, контроль разработки.
Документация Confluence, Notion, Google Docs PRD, требования, discovery notes, решения команды.
Прототипирование Figma, Miro Создание прототипов, CJM, пользовательских сценариев.
Аналитика SQL, Amplitude, Mixpanel, Google Analytics Retention, воронки, когортный анализ, проверка гипотез.
BI и отчёты Looker, Metabase, Power BI Дашборды, KPI, продуктовые метрики.
Коммуникация Slack, Teams, Zoom Синхронизация команды и работа со стейкхолдерами.

Какие артефакты создаёт Product Manager

Артефакт Инструменты Назначение
PRD Notion, Confluence Описание проблемы, решения, метрик и требований.
Roadmap Jira, Productboard, Notion План развития продукта.
Backlog Jira, Linear Список задач и инициатив.
Dashboard BI-системы, Amplitude Контроль продуктовых показателей.
User Research Miro, Figma, Docs Интервью, CJM, анализ пользователей.

Пример стека Product Manager

Уровень Инструменты
Начинающий PM Notion, Google Sheets, Jira, Figma, Google Analytics
Middle PM Jira, Confluence, SQL, Amplitude, Miro, BI
Senior PM Productboard, Data Warehouse, BI, эксперименты, аналитические платформы
Важно: инструменты сами по себе не делают продукт успешным. Главное — умение правильно формулировать проблемы, проверять гипотезы и принимать решения.

FAQ — инструменты Product Manager

Для старта достаточно Jira, Notion, Google Sheets, Figma и базовых инструментов аналитики.

Нет, но понимание SQL, данных и технических ограничений значительно помогает в работе.

Главный инструмент PM — это не программа, а процесс принятия решений на основе пользователей и данных.
