Специальные символы, якори, ссылки и комментарии в HTML

В HTML есть множество инструментов для оформления, навигации и комментирования веб-страниц. На этой странице вы узнаете:

  • Что такое специальные символы и как их использовать для корректного отображения текста;
  • Как создавать якори для переходов внутри страницы;
  • Как оформлять и управлять ссылками на другие страницы и документы;
  • Как добавлять комментарии — заметки для себя и других разработчиков;
  • Полезные советы для чистого, доступного и безопасного кода.

Эта информация пригодится каждому, кто работает с HTML — от новичков до профессионалов.

1. Специальные символы в HTML

Специальные символы — это символы, которые невозможно напрямую написать в HTML-коде, либо они имеют специальное значение в языке (например, угловые скобки < и > ). Для их отображения используются специальные коды — HTML-сущности .

Если вы напишете < или > в тексте, браузер воспримет их как начало или конец тега. Чтобы вывести эти символы именно как текст, используйте соответствующую сущность: &lt; для «меньше», &gt; для «больше» и т.д.

Также есть символы, которых нет на клавиатуре: пробелы, валюты, авторское право, математические знаки. Они добавляются с помощью сущностей.

Пример использования специальных символов:

Символ HTML-сущность Код (числовой) Описание
  &nbsp; &#160; Неразрывный пробел (для создания отступа внутри строки)
< &lt; &#60; Знак «меньше», используется для отображения символа <
> &gt; &#62; Знак «больше», используется для отображения символа >
& &amp; &#38; Амперсанд (и), используется для отображения символа &
© &copy; &#169; Символ авторского права
€ &euro; &#8364; Символ евро

Полный список спецсимволов: HTML Entities Reference

Пример использования:

<p>Выражение: 5 &lt; 10 &amp;&nbsp;7 &gt; 4</p>

Результат: Выражение: 5 < 10 7 > 4

2. Якоря в HTML

Якорь — это специальная метка внутри страницы, позволяющая быстро перейти к нужной области. Для этого используется тег <a> с атрибутом href="#имя_якоря" и элемент с соответствующим id .

Это удобно для длинных статей, FAQ, оглавлений или навигации по разделам.

Пример якоря:


<a href="#section1">Перейти к Разделу 1</a>

...

<h2 id="section1">Раздел 1</h2>

3. Ссылки в HTML

Ссылка — это элемент, который позволяет переходить на другие страницы, сайты, файлы или разделы внутри документа. Для создания ссылки используется тег <a> .

Основные атрибуты:

  • href : путь к странице, документу или якорю (например, href="https://site.com" , href="#section" )
  • target : способ открытия ссылки ( _blank — в новой вкладке, _self — в текущей)
  • title : всплывающая подсказка при наведении

Пример:


	<a href="https://www.example.com" target="_blank" title="Перейти на Example.com">Открыть Example.com</a>
Открыть Example.com

Внутренние ссылки (на якорь):


<a href="#tips">Полезные советы</a>
Полезные советы

4. Комментарии в HTML

Комментарий — это текст, который виден только в исходном коде страницы. Комментарии помогают разработчикам пояснять структуру, оставлять подсказки или отключать части кода без удаления.

Комментарии не отображаются в браузере для пользователей.

Пример комментария:


<!-- Это комментарий -->

<p>Этот текст виден, а комментарий выше — нет.</p>

Этот текст виден, а комментарий выше — нет.

Совет: Используйте комментирование для объяснения неочевидных решений, TODO и временного отключения кода.

5. Советы

  • Используйте специальные символы для корректного отображения текста (например, < , > , & ).
  • Для навигации по длинной странице используйте якори и уникальные id .
  • Оформляйте ссылки понятно: используйте title , target="_blank" для внешних сайтов, а для внутренней навигации — якори.
  • Комментируйте важные участки кода, чтобы ускорить командную работу и облегчить поддержку сайта.
  • Старайтесь писать код чисто и понятно, избегайте ненужных спецсимволов и неочевидных решений.
