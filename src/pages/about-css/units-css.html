Единицы Измерения в CSS

В CSS используются различные единицы измерения для задания размеров, длины, высоты, отступов и других свойств. Знание этих единиц помогает создавать адаптивные и масштабируемые веб-дизайны.

1. Абсолютные Единицы

Абсолютные единицы имеют фиксированные размеры, которые не зависят от других элементов или свойств.

Примеры:

  • px (пиксели) — базовая единица измерения на экране.
  • pt (пункты) — используется в печати (1pt = 1/72 дюйма).
  • cm (сантиметры) — не рекомендуется для экранов.

Таблица абсолютных единиц:

Единица Символ Описание
Пиксель px Основная единица измерения, фиксированная длина.
Пункт pt Используется в печатной графике.
Сантиметр cm Используется для физического измерения.

2. Относительные Единицы

Относительные единицы измеряются относительно других элементов, что делает их более гибкими в адаптивных дизайнах.

Примеры:

  • em — относительное измерение, основанное на размере шрифта родительского элемента.
  • rem — относительное измерение, основанное на размере шрифта корневого элемента (обычно <html> ).
  • % — процентное значение относительно родительского элемента.

Таблица относительных единиц:

Единица Символ Описание
em em Относительно размера шрифта родителя.
rem rem Относительно размера шрифта корневого элемента.
Процент % От относительного размера родителя.
vh vh Процент от высоты viewport (области видимости браузера).
vw vw Процент от ширины viewport.

3. Примеры Использования

Пример 1: Использование px


h1 {
    font-size: 24px; /* размер шрифта в пикселях */
}

Пример 2: Использование em


p {
    font-size: 1.5em; /* 150% от родительского шрифта */
}

Пример 3: Использование rem


.container {
    padding: 2rem; /* 2 раза размер шрифта корня */
}

Заключение

Знание и правильное использование единиц измерения в CSS является важной частью веб-разработки, позволяя создавать адаптивные и масштабируемые дизайны. Рекомендуется использовать относительные единицы для большей гибкости и доступности.
