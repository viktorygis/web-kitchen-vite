🔍 Что такое псевдоэлементы?

Псевдоэлементы — это специальные ключевые слова в CSS, которые добавляют стили к определённой части элемента или создают дополнительное содержимое, не меняя HTML-разметку.

  • Начинаются с двойного двоеточия :: (например, ::before ), но для совместимости часто поддерживается и одно двоеточие.
  • Позволяют стилизовать первую букву, первую строку, добавлять контент до или после элемента и многое другое.
  • Используются для украшения, создания декоративных элементов, выделения или вспомогательного оформления.

Пример

Добавим слово "Внимание:" перед параграфом с помощью ::before : 


p.warning::before {
  content: "Внимание: ";
  color: red;
  font-weight: bold;
}

Это сообщение с псевдоэлементом.

::before

::before позволяет добавить виртуальный элемент перед содержимым тега. Это удобно для добавления иконок, пометок или декоративных символов без изменения HTML-разметки. 

HTML:
	<p>Пример текста</p>

	CSS:
	p::before {
	  content: "🔥 ";
	  color: red;
	}

Пример текста

  • ::before — добавляет содержимое до текста элемента.
  • Часто используется для декоративных иконок, лейблов и т.д.

::after

::after работает аналогично ::before , но добавляет виртуальный элемент после содержимого тега. Это часто используется для добавления иконок, декоративных символов или пояснений без изменения HTML-кода. 

HTML:
<p>Пример текста</p>

CSS:
p::after {
   content: " ✨";
   color: gold;
}

Пример текста

  • ::after — добавляет содержимое после текста элемента.
  • Часто используется для декоративных целей: иконки, хвостики, стрелки, подписи и т.д.

::first-line

::first-line позволяет стилизовать первую строку текста в блочном элементе (например, абзаце или заголовке).

Это удобно для выделения начала абзаца, цитаты или привлечения внимания к первой строке текста.

💡 Работает только с блочными элементами и зависит от ширины контейнера. Если текст переносится, первая строка меняется. 

HTML:
<p>Прекрасное утро!
Пусть оно принесёт вам радость и вдохновение.</p>

CSS:
p::first-line {
   font-size: 1.5em;
   color: navy;
   font-style: italic;
}

Прекрасное утро! Пусть оно принесёт вам радость и вдохновение.

  • Стили применяются только к первой строке текста (до первого переноса строки).
  • Удобно для выделения зачинки или первой строки абзаца.

::first-letter

::first-letter позволяет стилизовать первую букву блочного элемента (например, абзаца или заголовка).

Это удобно для создания красивых заглавных букв, как в книгах и блогах.

💡 Работает только с блочными элементами. 

HTML:
<p>Начните день с улыбки!</p>

CSS:
p::first-letter {
   font-size: 2em;
   color: red;
   font-family: serif;
   font-weight: bold;
   float: left;
   line-height: 1;
   margin-right: 0.1em;
}

Начните день с улыбки!

  • Стили применяются только к первой букве блочного элемента.
  • Применяют для эффектных drop caps (больших букв в начале абзаца), оформления статей, блогов и книг.

::selection {

::placeholder Стилизует текст плейсхолдера в <input> и <textarea> .

input::placeholder {

}
::marker Стилизует маркеры списка.

li::marker {
