Flexbox

Flexbox (Flexible Box Layout) — это способ выстраивать элементы в строку или столбец с автоматическим распределением пространства.
Flexbox состоит из двух ключевых понятий:

  • Flex-контейнер — родительский элемент, внутри которого работает Flexbox ( display: flex ).
  • Flex-элементы — все дочерние элементы внутри контейнера, которые становятся гибкими.

Как включить Flexbox?

Чтобы включить Flexbox, задайте свойство display: flex для родительского элемента.
После этого все вложенные элементы автоматически становятся flex-элементами и располагаются в строку по умолчанию . 

.flex-container {
   display: flex; /* Включаем Flexbox */
}

Все элементы внутри контейнера становятся "гибкими" и располагаются в строку по умолчанию .
Вы можете управлять их размером, порядком и выравниванием с помощью дополнительных свойств Flexbox, таких как flex-direction , justify-content , align-items , flex-grow и другие.

Совет: Используйте display: flex для создания адаптивных, управляемых макетов без сложных float или inline-block стилей.

Направление Flexbox

Свойство flex-direction управляет направлением расположения flex-элементов.

Доступные значения:

  • row — элементы располагаются в строку (горизонтально, по умолчанию).
  • column — элементы располагаются в колонку (вертикально).
  • row-reverse — элементы располагаются в строку в обратном порядке.
  • column-reverse — элементы располагаются в колонку в обратном порядке. 

.flex-container {
  display: flex;
  flex-direction: row; /* row | column | row-reverse | column-reverse */
}

Перенос Flexbox

Свойство flex-wrap управляет тем, будут ли flex-элементы переноситься на новую строку/колонку, если не помещаются в контейнер.

Возможные значения:

  • nowrap — элементы не переносятся (по умолчанию).
  • wrap — элементы переносятся на новую строку или колонку.
  • wrap-reverse — элементы переносятся, но направление переноса обратное. 

.flex-container {
  display: flex;
  flex-wrap: wrap; /* nowrap | wrap | wrap-reverse */
}

Выравнивание Flexbox

Свойство align-items позволяет выравнивать элементы по перекрестной оси (перпендикулярно направлению flex-контейнера). 


.flex-container {
  align-items: center; /* или flex-start, flex-end, baseline, stretch */
}

Возможные значения:

  • flex-start — элементы прижаты к началу перекрестной оси.
  • center — элементы расположены по центру перекрестной оси.
  • flex-end — элементы прижаты к концу перекрестной оси.
  • baseline — элементы выровнены по их текстовой базовой линии.
  • stretch — элементы растягиваются на всю доступную высоту контейнера (по умолчанию).

Flex-grow

Свойство flex-grow определяет, насколько элемент может "расти" по сравнению с другими flex-элементами. 


.flex-item {
  flex-grow: 1; /* или 0, 2 и т.д. */
}

Order

Свойство order определяет визуальный порядок flex-элементов. Чем меньше значение — тем раньше элемент появляется в контейнере. 


.flex-item {
  order: 2; /* по умолчанию 0 */
}

Gap

Свойство gap добавляет отступы между flex-элементами, заменяя необходимость использования margin для внутренних отступов.
Важно: gap работает для flex-контейнеров во всех современных браузерах.

Можно использовать разные единицы измерения: px , em , % и др.
Также можно задать разные значения для строк и столбцов с помощью row-gap и column-gap . 


.flex-container {
  gap: 20px;           /* общий отступ между всеми элементами */
  row-gap: 30px;       /* отступ между строками (при переносе) */
  column-gap: 10px;    /* отступ между столбцами (при переносе) */
}
