Руководство по свойству background в CSS

Свойство background используется для установки фона элементов. Оно поддерживает различные параметры: цвет, изображение, позицию, повторение и другие.

Что такое background?

Свойство background в CSS используется для задания фона HTML-элементов.
Оно может включать цвет, изображение, позицию, повторение, размер, привязку и область отображения.

  • background-color — задаёт цвет фона
  • background-image — задаёт изображение фона
  • background-repeat — управляет повторением изображения
  • background-position — задаёт позицию фонового изображения
  • background-size — управляет размером фонового изображения
  • background-attachment — задаёт прокрутку фона (фиксированный или прокручивающийся)
  • background-clip — определяет область, в которой отображается фон
  • background-origin — задаёт область, откуда начинается фоновое изображение

Цвет фона

С помощью background-color можно задать однотонный фон для элемента. 


.bg-color {
  background-color: #ffe082;
}
Пример фона с цветом
background-color: #ffe082;

Фоновое изображение

Свойство background-image позволяет задать изображение как фон. 


.bg-image {
  background-image: url('example.jpg');
  background-size: cover;
  background-position: center;
}
Фон с изображением

Повторение фона

Свойство background-repeat управляет повтором фонового изображения. 


.bg-repeat {
  background-image: url('pattern.png');
  background-repeat: repeat;
}
Повторяющийся фон

Сокращённая запись

Можно объединить все свойства фона в одну строку с помощью background . 


.bg-shorthand {
  background: #a5d6a7 url('https://placehold.co/150x150') no-repeat right top;
}
Сокращённая запись background

Дополнительные свойства background

background-size

.bg-size { background-image: url('https://placehold.co/150x150'); background-size: contain; }
background-size: contain

background-attachment

.bg-attachment { background-image: url('https://placehold.co/150x150'); background-attachment: fixed; }
background-attachment: fixed

background-clip

.bg-clip { background: yellow; background-clip: content-box; padding: 20px; border: 5px solid red; }
background-clip: content-box

Практические примеры
