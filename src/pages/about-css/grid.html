Руководство по CSS Grid

Grid — это мощный способ создания двухмерных макетов.
В отличие от Flexbox, который работает в одном направлении (строка или столбец), Grid работает по двум осям одновременно.

Что такое CSS Grid?

CSS Grid — это система сеток, которая позволяет размещать элементы на странице по строкам и столбцам.
С её помощью легко создавать сложные и адаптивные макеты без лишнего кода.

  • Grid-контейнер — элемент, внутри которого создаётся сетка (например, <div class="grid-container"> ).
  • Grid-элементы — дочерние элементы контейнера, которые размещаются внутри сетки ( <div class="item"> ).

Пример: базовая сетка


<div class="grid-container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

<style>
.grid-container {
  display: grid; /* Включаем Grid */
}
</style>

Определение сетки

Свойства grid-template-columns и grid-template-rows определяют структуру сетки.
С их помощью можно задать количество и размеры столбцов и строк.

Пример 1: равные столбцы


.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}

Gap - Отступы между элементами

Для управления расстоянием между строками и столбцами используется свойство gap .
Оно удобно тем, что не требует писать дополнительные margin для каждого элемента.

Как работает gap

  • gap — задаёт одинаковое расстояние между всеми строками и столбцами.
  • row-gap — только между строками.
  • column-gap — только между столбцами.
  • Можно использовать любые единицы ( px , em , % ).

Пример: одинаковый gap между всеми ячейками


.grid-container {
  display: grid;
  grid-template-columns: 100px 100px 100px; /* Три столбца */
  gap: 20px; /* Расстояние между ячейками */
}

.item {
  background-color: coral;
  text-align: center;
  padding: 10px;
}

Автоматическое размещение элементов

Когда ты создаёшь сетку с помощью CSS Grid, необязательно вручную размещать каждый элемент.
Grid автоматически распределяет элементы по порядку в доступных ячейках — сверху вниз, слева направо.

Пример: авторазмещение элементов


<div class="grid-container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
</div>

<style>
.grid-container {
  display: grid;
  grid-template-columns: 100px 100px; /* Два столбца */
  grid-template-rows: auto auto;      /* Строки автоматически по содержимому */
}
.item {
  background-color: lightcoral;
  padding: 10px;
  text-align: center;
  border: 1px solid black;
}
</style>

  • .grid-container — создаёт сетку с 2 столбцами шириной по 100px.
    grid-template-rows: auto auto — высота строк определяется автоматически, в зависимости от содержимого.
  • .item — элементы автоматически размещаются в ячейках сверху вниз, слева направо.
    Grid не даст элементам выйти за пределы контейнера: создаст новые строки, если потребуется.

Явное размещение и растяжение элементов

Свойства grid-column и grid-row позволяют позиционировать элементы в сетке вручную и растягивать их на несколько строк или столбцов.

Пример размещения и растяжения


.grid-item {
  grid-column: 1 / 3; /* Элемент занимает столбцы 1 и 2 */
  grid-row: 1 / 2;    /* Элемент занимает первую строку */
}

  • grid-row — управляет положением и высотой элемента по строкам.
  • Можно "растягивать" элементы на несколько строк или столбцов.

Гибкое авторазмещение: auto-fill и minmax()

auto-fill и minmax() позволяют сделать сетку адаптивной и гибкой: столбцы автоматически добавляются, если хватает места, а их ширина ограничивается минимальным и максимальным значениями.

Пример гибкой сетки


.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
}

  • Сетка становится адаптивной: при изменении размера окна количество столбцов меняется автоматически.

Выравнивание элементов в Grid

Свойства justify-items и align-items управляют выравниванием содержимого внутри каждой ячейки.
justify-items — горизонтальное выравнивание.
align-items — вертикальное выравнивание.

Пример: центрирование содержимого ячеек


.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  justify-items: center;     /* Горизонтальное выравнивание */
  align-items: center;       /* Вертикальное выравнивание */
  height: 200px;
}

Все элементы расположены по центру каждой ячейки — как по горизонтали, так и по вертикали.

  • justify-items принимает значения: start , end , center , stretch .
  • align-items принимает значения: start , end , center , stretch .
  • Можно использовать justify-self и align-self для выравнивания отдельных элементов.
