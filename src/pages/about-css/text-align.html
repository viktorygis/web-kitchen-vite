Выравнивание текста в CSS

Правильное выравнивание текста — ключевой аспект веб-дизайна, влияющий на читабельность и визуальную привлекательность контента. В CSS для управления выравниванием используются три основных свойства: text-align , text-align-last и vertical-align .

Горизонтальное выравнивание (text-align)

Свойство text-align контролирует горизонтальное выравнивание текста и встроенных элементов внутри блочного контейнера.

Основные значения

text-align: left
Текст выровнен по левому краю. Это стандартное выравнивание для большинства западных языков. Обеспечивает лучшую читаемость для длинных абзацев.

text-align: center
Центрированный текст создает симметричную композицию. Идеально подходит для заголовков, цитат и коротких текстовых блоков.

text-align: right
Выравнивание по правому краю используется реже. Применяется для подписей, дат или языков с письмом справа налево.

text-align: justify
Текст равномерно распределяется по ширине блока, создавая ровные края с обеих сторон. Может создавать большие пробелы между словами в узких колонках.

Дополнительные значения text-align

Значение Описание Поддержка
start Аналогично left для LTR и right для RTL Все современные браузеры
end Аналогично right для LTR и left для RTL Все современные браузеры
match-parent Наследует направление родителя, затем как start Chrome, Firefox, Edge

Пример использования

/* Базовые настройки выравнивания */
body {
    text-align: left; /* Значение по умолчанию */
}

h1, h2, h3 {
    text-align: center;
}

blockquote {
    text-align: justify;
    hyphens: auto; /* Автоматические переносы для justify */
}

.article-meta {
    text-align: right;
}

Рекомендации по горизонтальному выравниванию

Основной текст

Для основного контента используйте выравнивание по левому краю. Оно обеспечивает лучшую читаемость для длинных текстов.

Заголовки

Центрируйте заголовки для визуального выделения. Для длинных заголовков можно использовать выравнивание по левому краю.

Мобильные устройства

Избегайте justify-выравнивания на узких экранах, так как оно создает большие пробелы между словами.

Доступность

Выравнивание по ширине может затруднить чтение для людей с дислексией. Предоставляйте возможность выбора выравнивания.

text-align и text-align-last
Свойство Значение Описание Пример стиля Пример использования
text-align left Выравнивает текст по левому краю контейнера text-align: left; Этот текст выровнен по левому краю.
right Выравнивает текст по правому краю контейнера text-align: right; Этот текст выровнен по правому краю.
center Выравнивает текст по центру контейнера text-align: center; Этот текст выровнен по центру.
justify Выравнивает текст по ширине, добавляя пробелы между словами text-align: justify; Этот текст выравнен по ширине и будет занимать всю ширину контейнера. Обратите внимание на неравномерные пробелы между словами.
text-align-last start Выравнивает последний ряд текста в зависимости от направления текста (слева или справа) text-align-last: start; Последний ряд текста выровнен по началу.
end Выравнивает последний ряд текста по концу (справа или слева) text-align-last: end; Последний ряд текста выровнен по концу.

Выравнивание последней строки (text-align-last)

Свойство text-align-last определяет, как будет выровнена последняя строка текста в блоке, особенно когда используется text-align: justify .

Значения свойства

Значение Описание
auto По умолчанию (обычно как justify, если text-align: justify)
left Выравнивает последнюю строку по левому краю
right Выравнивает последнюю строку по правому краю
center Центрирует последнюю строку
justify Растягивает последнюю строку по ширине
start , end Зависит от направления текста

Пример использования

/* Выравнивание последней строки */
.justified-text {
    text-align: justify;
    text-align-last: center; /* Центрируем последнюю строку */
    hyphens: auto; /* Автоматические переносы */
}
Пример text-align-last: center
Этот текст выровнен по ширине, но последняя строка центрирована. Обратите внимание, как это создает более сбалансированный вид для коротких абзацев. Свойство особенно полезно для выравнивания текста в узких колонках или при работе с газетным стилем верстки.

Вертикальное выравнивание (vertical-align)

Свойство vertical-align управляет вертикальным позиционированием inline и inline-block элементов относительно их родительской строки или ячейки таблицы.

Основные значения

Baseline
Text
vertical-align: baseline

Элементы выравниваются по базовой линии родителя (значение по умолчанию)

Top
Text
vertical-align: top

Элементы выравниваются по верхнему краю строки

Middle
Text
vertical-align: middle

Элементы выравниваются по середине родителя

Bottom
Text
vertical-align: bottom

Элементы выравниваются по нижнему краю строки

Text-Top
Text
vertical-align: text-top

Элементы выравниваются по верху родительского шрифта

Text-Bottom
Text
vertical-align: text-bottom

Элементы выравниваются по низу родительского шрифта

Особенности vertical-align

  • Работает только с inline , inline-block и table-cell элементами
  • Часто используется для выравнивания иконок с текстом
  • В таблицах управляет выравниванием содержимого ячеек
  • Может принимать числовые значения (например, vertical-align: 2px )

Примеры использования

/* Выравнивание иконки с текстом */
.icon {
    vertical-align: middle;
    margin-right: 5px;
}

/* Выравнивание в таблицах */
td {
    vertical-align: middle;
}

/* Настройка позиции индексов */
.sup {
    vertical-align: super;
    font-size: smaller;
}

.sub {
    vertical-align: sub;
    font-size: smaller;
}

Рекомендации по вертикальному выравниванию

Иконки и текст

Для выравнивания иконок с текстом используйте vertical-align: middle с небольшим отступом.

Таблицы

В таблицах чаще всего используют vertical-align: middle для центрирования содержимого ячеек.

Формулы

Для математических формул используйте super и sub для верхних и нижних индексов.

vertical-align
Значение Описание Пример стиля Пример использования
top Выравнивает элемент по верхнему краю контейнера vertical-align: top; Наше слово сверху
baseline Выравнивает элемент по базовой линии текста vertical-align: baseline; Наше слово по базовой линии
