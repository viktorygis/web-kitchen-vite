Основы шаблонов и директив Vue

Шаблоны и директивы — ключевые концепции Vue.js, позволяющие создавать динамические и реактивные пользовательские интерфейсы. В этом руководстве мы рассмотрим, как использовать шаблоны для отображения данных и директивы для управления поведением элементов на странице.

Для чего нужны Шаблоны и директивы:

  • Шаблоны позволяют внедрять динамические данные в HTML с помощью интерполяции.
  • Директивы предоставляют специальные атрибуты для управления отображением, циклом и другими аспектами DOM.
  • Вместе они делают Vue мощным инструментом для создания интерактивных веб-приложений.

Основные понятия

  • Шаблон — HTML-разметка с возможностью внедрения динамических данных и логики Vue.
  • Директива — специальный атрибут в шаблоне, начинающийся с v- , который указывает Vue, как обрабатывать элемент
    (например, v-if, v-for, v-bind).
  • Интерполяция — внедрение переменных и выражений в HTML с помощью {{ }}
  • Реактивность — способность Vue автоматически обновлять DOM при изменении данных.
  • Компоненты — переиспользуемые блоки кода, которые могут содержать свои шаблоны и логику.
  • Data() — функция, возвращающая объект с реактивными данными для компонента или приложения Vue.

Шаблоны Vue: структура и синтаксис

Шаблоны Vue — это HTML-разметка, которая может содержать динамические данные и директивы Vue для управления поведением элементов.

Основные элементы шаблона Vue

  • Шаблоны должны быть обернуты в один корневой элемент (например, <div>).
  • Используйте двойные фигурные скобки {{ }} для интерполяции данных.
  • Директивы Vue начинаются с префикса v- и управляют поведением элементов.
  • Шаблоны могут содержать условные конструкции и циклы с помощью директив.

Пример шаблона Vue

<div id="app">
  <h1>Привет, {{ name }}!</h1>
  <a v-bind:href="website">Мой сайт</a>
  <ul>
    <li v-for="user in users" :key="user">{{ user }}</li>
  </ul>
</div>
<script>
  Vue.createApp({
    data() {
      return {
        name: "Алексей",
        website: "https://example.com",
        users: ["Аня", "Макс", "Иван"]
      }
    }
  }).mount("#app");
</script>
Объяснение кода:
  • {{ name }} — подставляет значение переменной name из data().
  • v-bind:href="website" — — дт связывает-с атрибут href с переменной website.
  • v-for="user in users" :key="user" — перебирает массив users и создаёт элемент для каждого значения.

Интерполяция: вывод переменных в HTML

Интерполяция - это способ внедрения динамических данных в HTML-шаблоны Vue с помощью двойных фигурных скобок {{ }}.
Позволяет отображать значения переменных и выражений прямо в разметке.

Пример интерполяции:

В этом примере заголовок отображает значение переменной name из data().

<h1>Имя: {{ name }}</h1>
Объяснение:
  • {{ name }} — синтаксис интерполяции для вывода значения переменной name.
  • При изменении значения name в data(), заголовок автоматически обновится на странице.

Пример полного приложения с интерполяцией:

<div id="app">
  <h1>Привет, {{ name }}!</h1>
</div>

<script>
  Vue.createApp({
    data() {
      return {
        name: "Алексей"
      }
    }
  }).mount("#app");
</script>
Объяснение кода:
  • Шаблон содержит заголовок с интерполяцией {{ name }}.
  • В data() определена переменная name со значением "Алексей".
  • При запуске приложения Vue автоматически подставит значение name в заголовок.

🚀 Интерполяция — простой и мощный способ отображения динамических данных в шаблонах Vue, обеспечивающий реактивность и удобство разработки.

Директивы Vue: управление поведением элементов

Директивы Vue — специальные атрибуты в шаблоне, начинающиеся с префикса v-, которые указывают Vue, как обрабатывать элементы.
Они позволяют добавлять динамическое поведение, условный рендеринг, циклы и многое другое.

Основные директивы Vue:

  • v-bind — связывает атрибуты HTML с данными из data().
  • v-if — условно рендерит элемент в DOM.
  • v-show — управляет видимостью элемента через CSS.
  • v-for — перебирает массив и отображает список элементов.
  • v-model — двустороннее связывание данных с элементами формы.

Для чего нужны:

  • Управление отображением элементов (например, условный вывод с помощью v-if).
  • Перебор массивов и создание списков с помощью v-for.
  • Связывание атрибутов HTML с данными из data() с помощью v-bind.
  • Двустороннее связывание данных с элементами формы через v-model.

Альтернативы:

  • Использование JavaScript для манипуляции DOM напрямую (менее эффективно и реактивно).
  • Другие фреймворки, такие как React или Angular, которые имеют свои собственные подходы к управлению состоянием и рендерингом.

🚀 Директивы — мощный инструмент для управления поведением элементов в шаблонах Vue, обеспечивающий реактивность и удобство разработки.
✅ Без директив работает vue? Нет, директивы являются неотъемлемой частью Vue.js и обеспечивают его реактивность и динамическое поведение.

Директива v-model: двустороннее связывание данных

v-model — директива для двустороннего связывания данных с элементами формы. Она автоматически обновляет значение переменной в data(), когда пользователь изменяет данные в поле ввода.

Пример использования:

В этом примере поле ввода связано с переменной title, а заголовок отображает её значение.

<input v-model="title" type="text">
<h1>{{ title }}</h1>
Объяснение кода:
  • v-model="title" — двустороннее связывание данных для ввода заголовка.
  • {{ title }} — интерполяция для отображения введённого заголовка.
  • При изменении значения title в поле ввода, заголовок автоматически обновляется на странице.
🚀 v-model — мощный инструмент для двустороннего связывания данных с элементами формы, обеспечивающий реактивность и удобство разработки.

Директива v-bind: динамические атрибуты

v-bind — директива для связывания атрибутов HTML с данными из data().
Позволяет динамически изменять значения атрибутов на основе реактивных данных.
Атрибуты HTML - это свойства элементов, такие как href, src, class и т.д.

Пример использования:

В этом примере ссылка динамически связывается с переменной website из data() связывается с атрибутом href с помощью директивы v-bind. 

//Пример с v-bind в шаблоне
<a v-bind:href="website">Сайт</a>

//Пример с v-bind в data()
<script>
  Vue.createApp({
    data() {
      return {
        website: "https://example.com"
      }
    }
  }).mount("#app");
</script>
Объяснение кода:
  • v-bind:href="website" — связывает атрибут href с переменной website.
  • При изменении значения website в data(), ссылка автоматически обновится на странице.
  • Короткая запись: вместо v-bind:href можно использовать просто :href.
🚀 v-bind — мощный инструмент для динамического управления атрибутами HTML в шаблонах Vue, обеспечивающий реактивность и удобство разработки.

Директивы v-if и v-show: условный вывод элементов

v-if и v-show — директивы для условного отображения элементов на странице в зависимости от значений переменных из data() .

v-if - условный рендеринг

v-if добавляет или удаляет элемент из DOM в зависимости от условия. Если условие истинно, элемент отображается; если ложно — удаляется из DOM.

Пример:

<p v-if="isVisible">Этот текст появится, если isVisible = true</p>
Объяснение:
  • v-if="isVisible" — условие для отображения элемента. Если isVisible истинно, элемент отображается.
  • Если isVisible ложно, элемент удаляется из DOM.

v-show - управление видимостью

v-show управляет видимостью элемента с помощью CSS-свойства display . Элемент всегда присутствует в DOM, но скрывается или показывается в зависимости от условия. 

<p v-show="isVisible">Этот текст появится, если isVisible = true</p>

Различие между v-if и v-show

  • v-if удаляет элемент из DOM при ложном условии, а v-show скрывает его с помощью CSS.
  • v-if лучше использовать для редко отображаемых элементов, а v-show — для часто переключаемых элементов.
🚀 v-if и v-show — мощные директивы для условного отображения элементов в Vue, позволяющие создавать динамические и интерактивные интерфейсы.

Директива v-for: вывод списков из массива

v-for - директива для перебора массивов и объектов, позволяющая создавать списки элементов на основе данных из data().
Каждый элемент списка создаётся динамически для каждого значения в массиве или объекте.

Описание синтаксиса: v-for="item in items", где item — текущий элемент, а items — массив или объект для перебора.

Пример использования:

<ul>
   <li v-for="user in users" :key="user">{{ user }}</li>
</ul>

data() { return { users: ["Аня", "Макс", "Иван"] } }
Объяснение кода:
  • v-for="user in users" — перебирает массив users и создаёт элемент <li> для каждого значения.
  • :key="user" — уникальный ключ для каждого элемента списка, необходимый для оптимального обновления DOM.
  • {{ user }} — выводит текущее значение элемента из массива.
Внимание! У каждого элемента списка должен быть свой уникальный :key . Это помогает Vue правильно отслеживать и обновлять элементы при изменении массива.

Практика: мини-приложение с шаблоном и директивами

Создадим простое приложение Vue, которое использует шаблоны и директивы для отображения списка пользователей и управления видимостью элементов.

В этом примере мы применим:

  • Интерполяцию для отображения имени пользователя.
  • Директиву v-for для вывода списка пользователей.
  • Директиву v-if для условного отображения сообщения.
  • Директиву v-model для двустороннего связывания данных с полем ввода.

Пример приложения:

Привет, {{ name }}!

  • {{ user }}

Это условное сообщение!

Пример кода:

<div id="app">
  <input v-model="name" placeholder="Ваше имя">
  <h2>Привет, {{ name }}!</h2>
  <ul>
    <li v-for="user in users" :key="user">{{ user }}</li>
  </ul>
  <button @click="isVisible = !isVisible">Показать/скрыть приветствие</button>
  <p v-if="isVisible">Это условное сообщение!</p>
</div>

<script>
const app = Vue.createApp({
  data() {
    return {
      name: "Аня",
      users: ["Аня", "Макс", "Иван"],
      isVisible: true
    }
  }
});
app.mount('#app');
</script>
Объяснение кода:
  • v-model="name" — двустороннее связывание данных для ввода имени.
  • {{ name }} — интерполяция для отображения введённого имени.
  • v-for="user in users" — перебор массива пользователей и создание списка.
  • @click="isVisible = !isVisible" — обработчик события для переключения видимости сообщения.
  • v-if="isVisible" — условный вывод сообщения в зависимости от значения isVisible.
🚀 Это мини-приложение демонстрирует основные возможности шаблонов и директив Vue для создания динамических и интерактивных интерфейсов.

Советы и типичные ошибки

  • Всегда используйте :key для элементов в v-for — это предотвращает ошибки и баги при обновлении списка.
  • В data() обязательно возвращайте объект: return { ... } .
  • Не забывайте, что v-if удаляет элемент из DOM, а v-show только скрывает его.
  • Для динамических атрибутов используйте v-bind или короткую запись : .
Частая ошибка: отсутствие :key в списках — Vue может неправильно обновлять DOM при изменении массива.

Лучшие практики

Следуйте этим рекомендациям для эффективного использования шаблонов и директив Vue:

  • Держите шаблоны простыми и понятными — избегайте излишней вложенности.
  • Используйте компоненты для переиспользуемых частей интерфейса.
  • Проверяйте наличие уникальных ключей в v-for .
  • Используйте v-if для редко изменяемых элементов и v-show для часто переключаемых.
  • Комментируйте сложные участки шаблонов для лучшего понимания.

Реальные примеры использования шаблонов и директив во Vue

Пример 1. Реактивный заголовок с интерполяцией и v-model

В этом примере заголовок автоматически меняется при вводе текста. Для этого используется директива v-model и интерполяция.

Введите текст в поле ниже, и заголовок обновится автоматически.

Заголовок: {{ title }}

Код:

<input type="text" v-model="title" placeholder="Введите заголовок">
<h2>Заголовок: {{ title }}</h2>

data() {
  return {
    title: ""
  }
}
Объяснение кода:
  • v-model="title" — двустороннее связывание данных для ввода заголовка.
  • {{ title }} — интерполяция для отображения введённого заголовка.
  • При изменении значения title в поле ввода, заголовок автоматически обновляется на странице.

Пример 2. Динамическая ссылка с v-bind

Ссылка обновляется в зависимости от введённого URL. Для этого используется директива v-bind или её сокращённая запись :.

Введите URL сайта в поле ниже:

Перейти на сайт

Код:

<input type="text" v-model="website" placeholder="URL сайта">
<a :href="website">Перейти на сайт</a>

data() {
  return {
    website: "https://vuejs.org"
  }
}
Объяснение кода:
  • v-model="website" — двустороннее связывание данных для ввода URL сайта.
  • :href="website" — связывает атрибут href с переменной website.
  • При изменении значения website в поле ввода, ссылка автоматически обновляется на странице.

Пример 3. Переключение видимости с v-if и v-show

Текст появляется или скрывается по состоянию переменных. Используется директивы v-if и v-show для управления отображением элементов. v-if удаляет элемент из DOM, а v-show просто скрывает его с помощью CSS.

Нажмите кнопки ниже, чтобы показать или скрыть текст.

Этот текст показывает v-if!

Этот текст показывает v-show!

Код:

<button @click="isVisible = !isVisible">Показать/Скрыть с v-if</button>
<p v-if="isVisible">Этот текст показывает v-if!</p>
<button @click="show = !show">Показать/Скрыть с v-show</button>
<p v-show="show">Этот текст показывает v-show!</p>

data() {
  return {
    isVisible: true,
    show: true
  }
}
Объяснение кода:
  • @click="isVisible = !isVisible" — обработчик события для переключения видимости текста с помощью v-if.
  • v-if="isVisible" — условный вывод текста в зависимости от значения isVisible.
  • @click="show = !show" — обработчик события для переключения видимости текста с помощью v-show.
  • v-show="show" — управление видимостью текста с помощью CSS в зависимости от значения show.

Пример 4. Вывод массива с v-for

Список пользователей автоматически обновляется при изменении массива. Используется директива v-for для перебора массива и создания элементов списка.

Добавьте нового пользователя в список:

  • {{ user }}

Код:

<input type="text" v-model="newUser" placeholder="Добавить пользователя">
<button @click="addUser">Добавить</button>
<ul>
  <li v-for="user in users" :key="user">{{ user }}</li>
</ul>
data() { return { users: ["Аня", "Макс", "Иван"], newUser: "" } }
methods: {
  addUser() { if(this.newUser) { this.users.push(this.newUser); this.newUser = ""; } }
}
Объяснение кода:
  • v-model="newUser" — двустороннее связывание данных для ввода имени нового пользователя.
  • @click="addUser" — обработчик события для добавления нового пользователя в массив.
  • v-for="user in users" — перебор массива пользователей и создание списка.
  • :key="user" — уникальный ключ для каждого элемента списка.
