Стрелочные функции, основы

Стрелочные функции — компактный современный синтаксис для создания функций.
Они особенно удобны для коротких действий и колбэков.

  • Стрелочная функция: (аргументы) => выражение
  • Является сокращением для function -выражения
  • Часто используется для простых однострочных функций

Базовый синтаксис

Стрелочная функция записывается так:
let func = (arg1, arg2, ...argN) => expression;

Это эквивалентно:
let func = function(arg1, arg2, ...argN) { return expression; };

Пример:

let sum = (a, b) => a + b;

// То же самое:
// let sum = function(a, b) { return a + b; }

alert( sum(1, 2) ); // 3

Один аргумент — ещё короче

Если аргумент один, скобки можно опустить:

let double = n => n * 2;
// То же что: let double = function(n) { return n * 2 }

alert( double(3) ); // 6

Без аргументов

Если аргументов нет — скобки всё равно нужны, но они пустые:

let sayHi = () => alert("Hello!");

sayHi();

Динамическое создание стрелочных функций

Можно использовать стрелочные функции для динамического выбора действия:

let age = prompt("Сколько Вам лет?", 18);

let welcome = (age < 18) ?
  () => alert('Привет!') :
  () => alert("Здравствуйте!");

welcome();

Многострочные стрелочные функции

Если нужно написать несколько инструкций — используйте фигурные скобки.
В таком случае return нужно писать явно! 

let sum = (a, b) => {
  let result = a + b;
  return result;
};

alert( sum(1, 2) ); // 3

Итого

  • Стрелочные функции — удобный и краткий синтаксис для функций.
  • Однострочные: (...args) => выражение — сразу возвращают результат.
  • Многострочные: (...args) => { ... } — требуют return для возврата.
  • Используйте для коротких действий и колбэков.
