Взаимодействие с пользователем

Модальные окна — простой способ взаимодействия с пользователем в браузере.
JavaScript предоставляет три основные функции: alert , prompt и confirm .

Общие особенности:

  • Блокируют выполнение скрипта
  • Останавливают рендеринг страницы
  • Внешний вид зависит от браузера

Когда использовать:

  • Быстрое прототипирование
  • Отладка кода
  • Простая валидация

Функция alert

alert — показывает сообщение и ждёт, пока пользователь нажмёт кнопку «ОК».

Синтаксис:

alert(message);

Где message — текст для отображения.

Пример:

alert("Добро пожаловать на наш сайт!");

Особенности alert:

  • Окно имеет только кнопку OK
  • Возвращает undefined
  • Текст автоматически преобразуется в строку

Практическое применение:

  • Вывод важных уведомлений
  • Отображение сообщений об ошибках
  • Быстрая проверка значений при отладке

Функция prompt

prompt — отображает диалоговое окно с запросом на ввод текста.

Синтаксис:

result = prompt(title, [default]);

title — текст для отображения
default — начальное значение (необязательно)

Пример:

let name = prompt("Как вас зовут?", "Гость");
alert(`Привет, ${name}!`);

Особенности prompt:

  • Возвращает строку или null (если отмена)
  • В IE всегда указывайте второй параметр
  • Всегда проверяйте возвращаемое значение

Практическое применение:

  • Запрос имени пользователя
  • Ввод числовых значений
  • Простая авторизация

Пример с проверкой:

let age;
while (true) {
  age = prompt("Сколько вам лет?", 18);
  if (age === null) {
    alert("Отменено");
    break;
  }
  if (!isNaN(age)) break;
  alert("Введите число!");
}

Функция confirm

confirm — отображает диалоговое окно с вопросом и кнопками OK/Отмена.

Синтаксис:

result = confirm(question);

question — текст вопроса

Пример:

let isAdult = confirm("Вам есть 18 лет?");
alert(isAdult ? "Доступ разрешен" : "Доступ запрещен");

Особенности confirm:

  • Возвращает true (OK) или false (Отмена)
  • Нельзя изменить текст кнопок
  • Часто используется в цепочках условий

Практическое применение:

  • Подтверждение действий
  • Согласие с условиями
  • Выбор варианта "Да/Нет"

Пример с обработкой:

if (confirm("Вы уверены, что хотите удалить этот файл?")) {
  // Действие при подтверждении
  alert("Файл удален");
} else {
  // Действие при отмене
  alert("Удаление отменено");
}
