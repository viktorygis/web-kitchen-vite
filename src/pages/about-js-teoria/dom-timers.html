Таймеры в JavaScript: setTimeout и setInterval

Таймеры — это способ запускать действия через заданное время или с определённой периодичностью.
С их помощью сайты становятся динамичнее, появляются анимации, уведомления, автосохранения и многое другое!

Что можно делать с таймерами:

Отложить действие

setTimeout() позволяет выполнить код через заданное время.
Например: показать сообщение через 3 секунды.

Запускать действие периодически

setInterval() повторяет действие через равные промежутки времени.
Например: выводить "Тик!" каждую секунду.

Остановить таймер

clearInterval() и clearTimeout() — остановка таймера.
Например: остановить счетчик через 5 секунд.

Динамика и UX

Таймеры — основа для лоадеров, автосохранения, слайдеров, всплывающих подсказок и других интерактивных фишек!

setTimeout — запуск действия через время

setTimeout — функция для однократного отложенного действия.
setTimeout(fn, ms) — выполнит fn через ms миллисекунд.

Пример: показать сообщение через 3 секунды

setTimeout(() => {
  console.log("Привет, я появился через 3 секунды!");
}, 3000);

Подробно о setTimeout:

  • setTimeout — задаёт таймер для выполнения кода.
  • В скобках () — передаётся функция, которую нужно выполнить.
  • Внутри стрелочной функции — весь нужный код, который должен сработать через указанное время.
  • Время указывается в миллисекундах (1000 мс = 1 секунда).

Практика: setTimeout

setInterval — повторять действие через равные интервалы

setInterval — функция для повторного запуска действия.
setInterval(fn, ms) — будет выполнять fn каждое ms миллисекунд.

Пример: выводить сообщение каждую секунду

setInterval(() => {
  console.log("Привет каждую секунду!");
}, 1000);
  • Первый аргумент — функция для выполнения.
  • Второй аргумент — интервал в миллисекундах.
  • Функция будет запускаться снова и снова, пока её не остановить!

Практика: setInterval

Остановка таймера: clearInterval и clearTimeout

setInterval и setTimeout возвращают ID таймера.
Чтобы остановить таймер, используется clearInterval(id) или clearTimeout(id) .

Пример: остановить счётчик через 5 секунд

const intervalId = setInterval(() => {
  console.log("Тик!");
}, 1000);

setTimeout(() => {
  clearInterval(intervalId);
  console.log("Счётчик остановлен.");
}, 5000);
  • clearInterval требует ID таймера , который возвращает setInterval.
  • Так можно автоматически остановить любые циклические действия!
