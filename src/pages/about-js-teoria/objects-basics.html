Объекты в JavaScript

Объекты — это составные типы данных, представляющие собой коллекции пар "ключ-значение". Они используются для хранения сложных структур данных и являются основой объектно-ориентированного программирования в JavaScript.

Объекты же используются для хранения коллекций различных значений и более сложных сущностей. В JavaScript объекты используются очень часто, это одна из основ языка.

Объект может быть создан с помощью фигурных скобок {…} с необязательным списком свойств.
Свойство – это пара «ключ: значение» , где ключ – это строка (также называемая «именем свойства»), а значение может быть чем угодно.

Пример объекта:

//user	- объект с разными свойствами и методом
const user = {
  //Свойства (ключ: значение)
  name: 'Андрей',
  //Можно использовать кавычки для ключей с пробелами
  age: 30,
  //Булево значение
  isAdmin: true,

  //Метод (функция в свойстве)
  greet() {
  //Используем this для доступа к свойствам объекта
    console.log(`Привет, меня зовут ${this.name}!`);
  }
};

Мы можем представить объект в виде ящика с подписанными папками. Каждый элемент данных хранится в своей папке, на которой написан ключ. По ключу папку легко найти, удалить или добавить в неё что-либо.

Особенности объектов:

  • Динамические - свойства можно добавлять/удалять
  • Ссылочный тип данных
  • Могут содержать методы (функции)
  • Имеют прототипы для наследования
  • Ключи - строки или символы
  • Значения - любого типа

Способы создания объектов:

Литерал объекта (рекомендуется)

Литерал объекта - самый распространённый и удобный способ создания объектов.
Позволяет создавать объекты с заданными свойствами и методами в компактном виде.

Синтаксис

const obj = {
  key: 'value',
  method() { /* ... */ }
};

При использовании литерального синтаксиса {...} мы сразу можем поместить в объект несколько свойств в виде пар «ключ: значение».
У каждого свойства есть ключ (также называемый «имя» или «идентификатор»). После имени свойства следует двоеточие ":", и затем указывается значение свойства. Если в объекте несколько свойств, то они перечисляются через запятую.

Конструктор Object

Конструктор Object - функция, которая создаёт новый объект. Обычно его не используют для создания объектов, так как есть более удобные способы.

Синтаксис

const obj = new Object();
obj.key = 'value';

3. Object.create()

Object.create() - создаёт новый объект с указанным прототипом. Полезен для создания объектов с наследованием.

Синтаксис

const prototype = { a: 1 };
const obj = Object.create(prototype);

4. Конструктор класса (ES6)

Конструктор класса - современный синтаксис для создания объектов и работы с наследованием. Позволяет определять шаблоны для создания похожих объектов.

Синтаксис

class Person {
  constructor(name) {
    this.name = name;
  }
}
const obj = new Person('Андрей');

5. const

const user = {} Объект, объявленный как константа, может быть изменён Объект.

Например:

const user = {
   name: "John"
};

user.name = "Pete"; // (*)

alert(user.name); // Pete

Может показаться, что строка (*) должна вызвать ошибку, но нет, здесь всё в порядке.
Дело в том, что объявление const защищает от изменений только саму переменную user, а не её содержимое.
Определение const выдаст ошибку только если мы присвоим переменной другое значение: user=....
user хранит ссылку на объект. И это не может быть изменено. Но содержимое объекта менять можно.

Резюме

  • Литерал объекта - компактный и удобный способ создания объектов с заданными свойствами и методами.
  • Конструктор Object - менее распространённый способ, обычно не используется для создания объектов.
  • Object.create() - позволяет создавать объекты с указанным прототипом, полезен для наследования.
  • Конструктор класса (ES6) - современный синтаксис для создания объектов и работы с наследованием.

    • Выбор способа зависит от задачи: для простых объектов лучше использовать литералы, для сложных структур с наследованием - классы или Object.create().

    Работа со свойствами объектов

    1. Доступ к свойствам

    Точечная нотация

    Для обращения к свойствам используется запись «через точку»

    const person = { name: 'Андрей' };

// Точечная нотация
console.log(person.name); // 'Андрей'

    Обратите внимание, что строка в квадратных скобках заключена в кавычки (подойдёт любой тип кавычек).

    Скобочная нотация

    Для свойств, имена которых состоят из нескольких слов, доступ к значению «через точку» не работает:

    let user = { "likes birds": true };
// Скобочная нотация
console.log(user['likes birds']); // 'true'

    Для динамических ключей

    const person = { name: 'Андрей' };
// Для динамических ключей
const key = 'name';
console.log(person[key]); // 'Андрей'

    2. Добавление/изменение свойств

    const car = { model: 'Tesla' };

// Добавление свойства
car.year = 2023;

// Изменение свойства
car.model = 'Model S';

console.log(car);
// { model: 'Model S', year: 2023 }

    3. Удаление свойств

    Для удаления свойства мы можем использовать оператор delete 

    const obj = { a: 1, b: 2, c: 3 };

delete obj.b;
console.log(obj); // { a: 1, c: 3 }

// Проверка существования
console.log('b' in obj); // false

    4. Короткие свойства (ES6)

    Короткие свойства это синтаксис, позволяющий создавать свойства объекта, используя только имя переменной, если имя свойства совпадает с именем переменной.

    Это сокращает запись и улучшает читаемость кода.

    const name = 'Андрей';
const age = 30;

// Вместо { name: name, age: age }
const person = { name, age };

console.log(person); // { name: 'Андрей', age: 30 }

    in оператор

    Проверяет наличие свойства в объекте

    • Проверяет собственные и унаследованные свойства
    • Лучше использовать, чем сравнение с undefined
    propertyName in object

    Пример:

    //Проверка наличия свойства
const car = { model: 'Tesla' };
console.log('model' in car); // true
console.log('year' in car); // false

    Пример:

    Код для суммирования всех зарплат и сохраните результат в переменной sum. Должно получиться 390.
    //Если объект salaries пуст, то результат должен быть 0. 

    let salaries = {
	    John: 100,
	    Ann: 160,
	    Pete: 130,
	};

	let sum = 0;

	for (let key in salaries) {
	    sum += salaries[key];
	}
	console.log(sum);

    Пример:

    Умножаем все числовые свойства на 2
    Создайте функцию multiplyNumeric(obj), которая умножает все числовые свойства объекта obj на 2. 

    let menu = {
    width: 200,
    height: 300,
    title: "My menu",
};

console.log(menu); //до вызова функции
//{width: 200, height: 300, title: 'My menu'}

function multiplyNumeric(obj) {
    for (let key in obj) {
        if (typeof obj[key] === "number") {
            obj[key] *= 2;
        }
    }
}

multiplyNumeric(menu);
console.log(menu); //после вызова функции
//{width: 400, height: 600, title: 'My menu'}

    Основные методы объектов

    1. Object.keys()

    Возвращает массив ключей объекта

    • Возвращает только перечисляемые свойства
    • Не возвращает унаследованные свойства
    Object.keys(obj)

    Пример:

    const person = {
   name: "Андрей",
   age: 30,
};
const keys = Object.keys(person);
console.log(keys); // ['name', 'age']

    2. Object.values()

    Возвращает массив значений объекта

    • Порядок значений соответствует Object.keys()
    • ES2017+ (не поддерживается в старых браузерах)
    Object.values(obj)

    Пример:

    const person = { name: 'Андрей', age: 30 };
const values = Object.values(person);
console.log(values); // ['Андрей', 30]

    3. Object.entries()

    Возвращает массив пар [ключ, значение]

    • Удобен для преобразования объектов в Map
    • Используется для итерации по объекту
    Object.entries(obj)

    Пример:

    const person = {
    name: "Андрей",
    age: 30,
};
const entries = Object.entries(person);
console.log(entries); // [['name', 'Андрей'], ['age', 30]]

// Итерация
for (const [key, value] of Object.entries(person)) {
    console.log(`${key}: ${value}`);
}

    4. Object.assign()

    Копирует свойства из объектов-источников в целевой объект

    • Изменяет целевой объект (мутирующий метод)
    • Не копирует унаследованные свойства
    • Поверхностное копирование (не глубокое)
    Object.assign(target, ...sources)

    Примеры:

    // Объединение объектов
const target = {
    a: 1,
};
const source = {
    b: 2,
};
Object.assign(target, source);
console.log(target); // { a: 1, b: 2 }

// Клонирование объекта (поверхностное)
//поверхностное почему? так как если в объекте есть вложенные объекты, то они будут скопированы по ссылке
const obj = {
    x: 10,
    y: 20,
};
//клонирование obj в новый пустой объект
const clone = Object.assign({}, obj);
console.log(clone); // { x: 10, y: 20 }

    5. Object.freeze()

    Замораживает объект, предотвращая изменения

    • Нельзя добавлять/удалять/изменять свойства
    • В строгом режиме попытки изменения вызовут ошибку
    • Не рекурсивен (вложенные объекты не замораживаются)
    Object.freeze(obj)

    Пример:

    const person = Object.freeze({ name: 'Андрей' });
person.name = 'Иван'; // Не сработает
console.log(person.name); // 'Андрей'

// Проверка
console.log(Object.isFrozen(person)); // true

    Продвинутые техники работы с объектами

    Деструктуризация объектов (ES6)

    const user = {
  id: 1,
  name: 'Андрей',
  address: {
    city: 'Москва'
  }
};

// Базовый вариант
const { name, id } = user;

// С псевдонимом
const { name: userName } = user;

// Вложенная деструктуризация
const { address: { city } } = user;

console.log(userName, city); // 'Андрей', 'Москва'

    Опциональная цепочка (ES2020)

    const user = {
  profile: {
    name: 'Андрей'
  }
};

// Безопасный доступ к вложенным свойствам
console.log(user?.profile?.name); // 'Андрей'
console.log(user?.contacts?.email); // undefined

// С вызовом методов
user.greet?.(); // Не вызовет ошибку, если метода нет

    Object.defineProperty()

    Определяет новое или изменяет существующее свойство

    • Позволяет задать дескрипторы свойства
    • Контроль перечислимости, записываемости и т.д.
    • Основа для геттеров/сеттеров
    Object.defineProperty(obj, prop, descriptor)

    Пример:

    const obj = {};

Object.defineProperty(obj, 'readOnly', {
  value: 42,
  writable: false,
  enumerable: true
});

obj.readOnly = 100; // Не изменится (в строгом режиме - ошибка)
console.log(obj.readOnly); // 42

    Геттеры и сеттеры

    const user = {
  firstName: 'Андрей',
  lastName: 'Иванов',

  // Геттер
  get fullName() {
    return `${this.firstName} ${this.lastName}`;
  },

  // Сеттер
  set fullName(value) {
    [this.firstName, this.lastName] = value.split(' ');
  }
};

console.log(user.fullName); // 'Андрей Иванов'
user.fullName = 'Петр Сидоров';
console.log(user.firstName); // 'Петр'
