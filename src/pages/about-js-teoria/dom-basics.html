Основы DOM в JavaScript

DOM (Document Object Model) — это интерфейс, который представляет структуру HTML-документа в виде дерева объектов и позволяет JavaScript взаимодействовать с содержимым страницы.

  • Управляй элементами: добавляй, удаляй, изменяй их динамически.
  • Делай страницы "живыми": обработка кликов, показ сообщений, анимации — всё это работа DOM.
  • Создавай сайты , которые реагируют на пользователя.

Что такое DOM?

DOM — это не HTML, а модель страницы. Когда браузер читает HTML, он создает из него дерево объектов.
Каждый тег становится узлом.
Вложенные элементы (например, <ul> с <li> ) — это "родители" и "дети". 

<body>
  <h1>Привет, мир!</h1>
  <p>Это параграф.</p>
</body>

В DOM это выглядит так:

body
├── h1
└── p
Важно: DOM — это не сам HTML, а структура, которую браузер создает для работы с ним.

DOM — это дерево

Корнем дерева всегда является <html>
Внутри него находятся <head> и <body>
Остальные элементы — "ветки" и "листья". 

<html>
  <head>
    <title>Заголовок страницы</title>
  </head>
  <body>
    <h1>Привет, мир!</h1>
    <p>Это параграф.</p>
  </body>
</html>

DOM-структура:

html
├── head
│   └── title
└── body
    ├── h1
    └── p
💡 DOM открывает доступ ко всему, что ты видишь на странице.
JavaScript может узнать текст в <h1> , изменить цвет кнопки, если на нее нажали, скрыть или удалить элемент.
