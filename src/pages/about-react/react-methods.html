Методы React-компонентов

Методы — функции внутри компонента.
Для side-эффектов и "жизненного цикла" используйте useEffect .
Для обработки событий — свои методы.
Все методы — обычные функции JS!

Жизненный цикл и useEffect

useEffect заменяет методы жизненного цикла из class-компонентов.
Для подписок, запросов, таймеров — используйте useEffect! 

React.useEffect(() => {
  // код выполнится при монтировании (аналог componentDidMount)
  return () => {
    // cleanup: выполнится при размонтировании
  }
}, []);

Пользовательские методы


function Counter() {
  const [count, setCount] = React.useState(0);
  function increment() {
    setCount(count + 1);
  }
  return <button onClick={increment}>{count}</button>;
}
Все методы — обычные функции, объявляйте их внутри компонента.

Практика: useEffect и методы


function Timer() {
  const [sec, setSec] = React.useState(0);
  React.useEffect(() => {
    const id = setInterval(() => setSec(s => s + 1), 1000);
    return () => clearInterval(id);
  }, []);
  return <div>Прошло секунд: {sec}</div>;
}

Советы и типичные ошибки

  • Не объявляйте методы вне компонента, если они используют state.
  • useEffect без зависимостей [] — только при монтировании.
  • Не забывайте про очистку (cleanup) внутри useEffect!

Лучшие практики

  • Для side-эффектов всегда используйте useEffect.
  • Давайте методам осмысленные имена: fetchData , handleClick .
  • Не дублируйте логику — выносите общие методы.

Примеры методов в React

useEffect для fetch


function Users() {
  const [users, setUsers] = React.useState([]);
  React.useEffect(() => {
    fetch("https://reqres.in/api/users?per_page=3").then(r => r.json()).then(json => setUsers(json.data));
  }, []);
  return <ul>{users.map(u => <li key={u.id}>{u.first_name} {u.last_name}</li>)}</ul>;
}
