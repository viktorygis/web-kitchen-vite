Основы React: кратко

React — библиотека для создания пользовательских интерфейсов. Главные концепции: компоненты, props, состояние, события.

Компоненты


function Hello() {
  return <h2>Привет, React!</h2>;
}
Каждый компонент — функция, возвращающая JSX.

Props (свойства)


function Welcome(props) {
  return <h2>Привет, {props.name}!</h2>;
}
<Welcome name="Аня" />
Props — способ передавать данные в компонент.

Состояние (state)


function Counter() {
  const [count, setCount] = React.useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
useState — хук для хранения и изменения состояния компонента.

События


<button onClick={handleClick}>Кликни меня</button>
События передаются как props (например, onClick, onChange).

Рендеринг


const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<App />);
Вся страница — дерево компонентов, рендерится через ReactDOM.

Советы

  • Именуйте компоненты с большой буквы.
  • Props — только для чтения.
  • Состояние — для динамики.
  • Стройте интерфейс из простых и переиспользуемых компонентов.
