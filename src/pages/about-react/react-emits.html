Callbacks и Events в React

Callbacks — это функции, которые мы передаем компонентам через props для обработки событий.
Events — стандартные события браузера (click, change, submit и др.), которые мы обрабатываем в компонентах.
Этот подход позволяет строить реактивные интерфейсы, где дочерние компоненты могут сообщать родителям о действиях пользователя.

Основы событий в React


function MyButton() {
  function handleClick() {
    alert('Клик!');
  }
  return <button onClick={handleClick}>Нажми меня</button>;
}
onClick — это prop для стандартного события click. Передайте функцию-обработчик!

Передача callbacks через props


function Child({ onAction }) {
  return <button onClick={onAction}>Выполнить действие</button>;
}
function Parent() {
  function handleAction() {
    alert('Действие выполнено!');
  }
  return <Child onAction={handleAction} />;
}
Родитель передает функцию-обработчик дочернему компоненту как props. Это стандартный паттерн связи между компонентами!

Как дочерний компонент сообщает родителю о событии?


function Counter({ onIncrement }) {
  return <button onClick={onIncrement}>+1</button>;
}
function App() {
  const [count, setCount] = React.useState(0);
  function increment() {
    setCount(c => c + 1);
  }
  return (
    <div>
      <p>Счетчик: {count}</p>
      <Counter onIncrement={increment} />
    </div>
  );
}
Дочерний компонент вызывает onIncrement , а родитель обновляет состояние.

Практика: форма обратной связи


<div id="root"></div>
<script type="text/babel">
function FeedbackForm({ onSend }) {
  const [text, setText] = React.useState('');
  function handleSubmit(e) {
    e.preventDefault();
    onSend(text);
    setText('');
  }
  return (
    <form onSubmit={handleSubmit}>
      <input value={text} onChange={e => setText(e.target.value)} placeholder="Ваш отзыв..." />
      <button type="submit">Отправить</button>
    </form>
  );
}

function App() {
  const [feedback, setFeedback] = React.useState([]);
  function addFeedback(newText) {
    setFeedback(arr => [...arr, newText]);
  }
  return (
    <div>
      <h3>Отзывы</h3>
      <FeedbackForm onSend={addFeedback} />
      <ul>{feedback.map((f, i) => <li key={i}>{f}</li>)}</ul>
    </div>
  );
}

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<App />);
</script>

Советы и типичные ошибки

  • Передавайте только необходимые функции-обработчики!
  • Не вызывайте callback сразу: onClick={handler} , а не onClick={handler()} .
  • Используйте стрелочные функции для передачи параметров: onClick={() => handler(value)} .
  • Не изменяйте props в дочернем компоненте!

Лучшие практики

  • Документируйте типы callbacks (PropTypes, JSDoc).
  • Давайте осмысленные имена обработчикам: onClick , onSubmit , onChange .
  • Используйте memoization для callback-функций с useCallback, если нужно.
  • Передавайте callbacks только вниз по дереву (от родителя к дочернему компоненту).
  • Для связи между несвязанными компонентами используйте Context или state management.

Примеры использования callbacks и событий

Передача параметра в callback


function DeleteButton({ id, onDelete }) {
  return <button onClick={() => onDelete(id)}>Удалить</button>;
}

Обработка событий формы


function LoginForm({ onLogin }) {
  function handleSubmit(e) {
    e.preventDefault();
    onLogin();
  }
  return (
    <form onSubmit={handleSubmit}>
      <button type="submit">Войти</button>
    </form>
  );
}

Callback для изменения состояния родителя


function Toggle({ onToggle }) {
  return <button onClick={onToggle}>Переключить</button>;
}
function App() {
  const [on, setOn] = React.useState(false);
  return (
    <div>
      <p>Состояние: {on ? 'Вкл' : 'Выкл'}</p>
      <Toggle onToggle={() => setOn(o => !o)} />
    </div>
  );
}
