Props в React

Props — способ передавать данные компонентам.
Любой компонент может принимать props (свойства) от родителя.
Props — только для чтения!

Основы props


function Welcome(props) {
  return <h2>Привет, {props.name}!</h2>;
}
В props приходит объект со всеми переданными свойствами.

Передача props


<Welcome name="Аня" />
Передавайте любые данные: строки, числа, функции, массивы!

Дочерние элементы: props.children


function Wrapper(props) {
  return <div className="wrap">{props.children}</div>;
}
<Wrapper>Контент внутри!</Wrapper>
props.children — всё, что написано внутри тега компонента.

Практика: карточка с props


<div id="root"></div>
<script type="text/babel">
function Card(props) {
  return <div style={{border:"1px solid #ccc", padding:10}}>
    <h3>{props.title}</h3>
    <div>{props.children}</div>
  </div>;
}
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(
  <Card title="Заголовок">Текст карточки</Card>
);
</script>

Советы и типичные ошибки

  • Props нельзя изменять внутри компонента!
  • Для передачи данных вниз по дереву — используйте props.
  • Используйте деструктуризацию для удобства: function Comp({title}) .

Лучшие практики

  • Документируйте props (PropTypes, JSDoc).
  • Используйте defaultProps или значения по умолчанию.
  • Не передавайте избыточные props.

Примеры использования props

Передача функции как props


function Button({onClick, children}) {
  return <button onClick={onClick}>{children}</button>;
}

Массив как props


function List({items}) {
  return <ul>{items.map(i => <li key={i}>{i}</li>)}</ul>;
}
