useReducer в React

useReducer — хук для управления сложным состоянием.
Подходит для форм, списков, сложных UI, где много вариантов изменения.

Основы useReducer


const [state, dispatch] = React.useReducer(reducer, initialState);
reducer — функция для изменения состояния.
dispatch — функция для отправки действий.

Использование useReducer


function reducer(state, action) {
  switch(action.type) {
    case "inc": return {count: state.count + 1};
    case "dec": return {count: state.count - 1};
    default: return state;
  }
}
function Counter() {
  const [state, dispatch] = React.useReducer(reducer, {count: 0});
  return (
    <div>
      <button onClick={() => dispatch({type: "inc"})}>+</button>
      <span>{state.count}</span>
      <button onClick={() => dispatch({type: "dec"})}>-</button>
    </div>
  );
}
Каждый dispatch отправляет action — объект с типом действия.

Практика: список задач на useReducer


<div id="root"></div>
<script type="text/babel">
function reducer(state, action) {
  switch(action.type) {
    case "add":
      return [...state, action.text];
    case "remove":
      return state.filter((_, i) => i !== action.idx);
    default:
      return state;
  }
}
function TodoApp() {
  const [tasks, dispatch] = React.useReducer(reducer, []);
  const [value, setValue] = React.useState("");
  return (
    <div>
      <input value={value} onChange={e => setValue(e.target.value)} />
      <button onClick={() => {dispatch({type:"add", text:value}); setValue("");}}>Добавить</button>
      <ul>
        {tasks.map((t,i) => <li key={i}>{t} <button onClick={() => dispatch({type:"remove", idx:i})}>Удалить</button></li>)}
      </ul>
    </div>
  );
}
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<TodoApp />);
</script>

Советы и типичные ошибки

  • useReducer удобен, когда много вариантов изменения состояния.
  • Держите reducer чистым — без побочных эффектов.
  • Action — всегда объект с типом.

Лучшие практики

  • Выносите reducer вне компонента для чистоты кода.
  • Документируйте типы actions и состояние.
  • Используйте useReducer вместо useState для сложных форм и списков.
