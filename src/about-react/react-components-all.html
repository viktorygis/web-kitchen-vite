Компоненты React: кратко

Компоненты — основа любого React-приложения. Позволяют разбивать интерфейс на независимые части и переиспользовать их.

Виды компонентов

  • Функциональные — обычные функции, возвращающие JSX.
  • Классовые — используются редко, в новых проектах предпочтительнее функции с хуками. 

function MyComp() { return <div>Текст</div>; }

Props — свойства компонента


function Button({children, onClick}) {
  return <button onClick={onClick}>{children}</button>;
}
Props позволяют делать компоненты гибкими и настраиваемыми.

Children — композиция компонентов


function Panel({children}) {
  return <section>{children}</section>;
}
<Panel><h2>Заголовок</h2></Panel>
props.children — всё, что внутри тега компонента.

Композиция


function Card({children}) {
  return <div className="card">{children}</div>;
}
Композиция — основной способ строить интерфейсы из простых блоков.

Пример: карточка


function Card({title, children}) {
  return <div>
    <h3>{title}</h3>
    <div>{children}</div>
  </div>;
}
<Card title="Заголовок">Контент</Card>

Советы

  • Делайте компоненты максимально независимыми и простыми.
  • Передавайте только нужные props.
  • Используйте composition для гибких интерфейсов.
  • Документируйте props и структуру компонентов.
