Запросы к API (fetch, axios) в React

fetch и axios — популярные способы взаимодействия с внешними API.
Обычно запросы выполняются внутри useEffect .

fetch — стандартный способ


React.useEffect(() => {
  fetch("https://jsonplaceholder.typicode.com/users")
    .then(res => res.json())
    .then(data => setUsers(data));
}, []);
fetch — встроенный способ сделать HTTP-запрос.

axios — удобная библиотека


import axios from "axios";
React.useEffect(() => {
  axios.get("https://jsonplaceholder.typicode.com/users")
    .then(res => setUsers(res.data));
}, []);
axios — удобнее для сложных запросов, поддерживает interceptors.

Асинхронные запросы с async/await


React.useEffect(() => {
  async function fetchUsers() {
    const res = await fetch("https://jsonplaceholder.typicode.com/users");
    const data = await res.json();
    setUsers(data);
  }
  fetchUsers();
}, []);
Для сложных запросов используйте async/await .

Практика: загрузка списка пользователей


<div id="root"></div>
<script type="text/babel">
function App() {
  const [users, setUsers] = React.useState([]);
  React.useEffect(() => {
    fetch("https://jsonplaceholder.typicode.com/users")
      .then(res => res.json())
      .then(data => setUsers(data));
  }, []);
  return (
    <div>
      <h3>Пользователи</h3>
      <ul>{users.map(u => <li key={u.id}>{u.name}</li>)}</ul>
    </div>
  );
}
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<App />);
</script>

Советы и типичные ошибки

  • Делайте запросы только внутри useEffect .
  • Обрабатывайте ошибки при запросе.
  • Отменяйте запросы при размонтировании компонента.

Лучшие практики

  • Выносите логику запросов в отдельные функции или хуки.
  • Документируйте структуру API для команды.
  • Используйте состояния loading/error.

Примеры запросов

POST-запрос с fetch


fetch("/api", {
  method: "POST",
  body: JSON.stringify({name:"Test"}),
  headers: {"Content-Type": "application/json"}
});

POST-запрос с axios


axios.post("/api", {name:"Test"});
