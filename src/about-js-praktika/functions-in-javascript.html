В этой секции мы рассмотрим функции, которые позволяют организовать код и уменьшить его дублирование.

Задача: Проверка возраста

Создайте функцию checkAge(age) , которая будет проверять возраст пользователя и возвращать true , если возраст больше 18, и в противном случае запрашивать разрешение у родителей. 


function checkAge(age) {
  if (age > 18) {
    return true; // Возраст больше 18, доступ разрешен
  } else {
    return confirm("Родители разрешили?"); // Запрашиваем разрешение у родителей
  }
}
