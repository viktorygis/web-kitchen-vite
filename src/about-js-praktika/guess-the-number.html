Угадай число

Я загадал число от 1 до 100. Попробуй угадать!

Давай сделаем интересный и красивый проект — игру "Угадай число". Это будет простой, но забавный проект, который поможет закрепить пройденый материал, используя объекты, методы объектов, стрелочные функции и шаблонные строки.

Описание игры:

  • Игра генерирует случайное число от 1 до 100.
  • Игрок должен угадать это число.
  • При каждом вводе программы сообщается, больше или меньше число игрока по сравнению с загаданным.
  • Игрок может снова попытаться, пока не угадает.

Структура файлов:

  • HTML — структура страницы (красивое оформление).
  • CSS — стили для страницы (красивый дизайн).
  • JavaScript (ES6) — логика игры (с использованием ES6).
Мы будем работать только с ES6, но вы также можете покопаться в HTML или CSS и исправить дизайн, как вам хочется. Только будьте аккуратнее! Если что-то меняете, учитывайте сразу все 3 файла, чтобы ничего не поломать.

1️⃣ Создание объекта game

Мы создаем объект game, который будет хранить:

  • Загаданное число secretNumber с null
  • Мы ещё не загадали число. Когда игра начинается, число генерируется случайным образом, и оно будет заменено на реальное число.
  • Количество попыток attempts будет равняться 0
  • Количество попыток равно нулю, потому что игрок ещё не сделал ни одной попытки. Каждый раз, когда начинается новая игра, попытки сбрасываются на 0.
  • Максимальное количество попыток maxAttempts давайте сделаем 10

2️⃣ Метод startGame

Этот метод запускает игру: генерирует случайное число, сбрасывает количество попыток и очищает сообщения на экране. Он вызывается при старте игры и после сброса.

Давай обратимся к методу startGame, который находится внутри объекта game
