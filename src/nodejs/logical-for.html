Цикл for в JavaScript

Цикл for — это конструкция JavaScript, которая выполняет блок кода повторно, пока выполняется условие.
Используется, когда заранее известно количество повторений.

Циклы for применяются для перебора элементов массива, выполнения действий заданное количество раз и работы с диапазонами чисел.

Где используется for

  • Перебор элементов массива.
  • Выполнение действия заданное количество раз.
  • Обработка диапазонов чисел.
  • Итерация по строкам, объектам и другим коллекциям.

Основные понятия

Цикл с известным количеством повторений — конструкция for проверяет условие, но в отличие от while всегда знает, сколько раз повторится блок кода.

  • Инициализация — начальное значение переменной-счётчика.
  • Условие — выражение, при котором цикл продолжается.
  • Инкремент — изменение счётчика после каждой итерации.
  • Тело цикла — блок кода, который выполняется при каждой итерации.

Синтаксис цикла for

Синтаксис for — структура записи цикла, состоящая из трёх частей и блока выполняемого кода.

Базовая структура:

for (инициализация; условие; инкремент) {
  // код выполняется пока условие true
}
  • Инициализация — выполняется один раз перед циклом.
  • Условие — проверяется перед каждой итерацией.
  • Инкремент — выполняется после каждой итерации.
  • Тело цикла — находится внутри фигурных скобок.

Простой пример:

for (let count = 1; count <= 5; count++) {
  console.log(count);
}
Как это работает:
  1. let count = 1 – создаём переменную-счётчик.
  2. count <= 5 – проверяем условие (если true, выполняем код).
  3. count++ – увеличиваем переменную после каждой итерации.
Когда использовать for
  • Когда заранее известно, сколько раз нужно повторить код.
  • Когда есть счётчик (например, перебор чисел или элементов массива).

Скобки цикла for

Если в теле цикла только одно действие, можно не писать фигурные скобки {}

for (let i = 1; i <= 3; i++) console.log(i);

⚠ Но если в for больше одной строки кода, скобки обязательны!

Как работает цикл for

Принцип работы for — цикл автоматически управляет счётчиком и проверяет условие перед каждой итерацией.

Пример со счётчиком:

for (let count = 1; count <= 5; count++) {
  console.log(count);
}

В этом примере цикл выполнится пять раз и выведет числа от 1 до 5.

Итерация Значение count Условие Действие
1 1 1 <= 5 → true Выполнить код, count++
2 2 2 <= 5 → true Выполнить код, count++
6 6 6 <= 5 → false Цикл завершается

Перебор массива через for:

const users = ["Anna", "Ivan", "Maria"];

for (let i = 0; i < users.length; i++) {
  console.log(users[i]);
}

В этом примере цикл перебирает элементы массива. Значение i используется как индекс для доступа к элементам.

Практические примеры

Практическое применение for — решение реальных задач обработки данных и работы с коллекциями.

Суммирование элементов массива

const numbers = [10, 20, 30, 40];
let sum = 0;

for (let i = 0; i < numbers.length; i++) {
  sum += numbers[i];
}

console.log("Сумма:", sum); // 100

Поиск элемента в массиве

const users = ["Anna", "Ivan", "Maria"];
const search = "Ivan";

for (let i = 0; i < users.length; i++) {
  if (users[i] === search) {
    console.log("Найден на позиции:", i);
    break; // Выход из цикла
  }
}

Создание таблицы умножения

for (let i = 1; i <= 10; i++) {
  console.log(`${i} × 5 = ${i * 5}`);
}

Обработка строки

const str = "Hello";

for (let i = 0; i < str.length; i++) {
  console.log(str[i]);
}

Цикл перебирает каждый символ строки и выводит его на экран.

for vs while

Сравнение for и while — выбор подходящего цикла зависит от того, известно ли заранее количество повторений.

Цикл Когда использовать Пример
for — количество повторений известно. Перебор массива, диапазон чисел, известное количество операций. for (let i = 0; i < 10; i++)
while — количество повторений неизвестно. Ожидание условия, обработка потока данных. while (condition)

Пример for:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Эквивалент с while:

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Оба варианта выполняют одно и то же, но for более компактен и удобен для известного количества повторений.

Типичные ошибки при работе с for

Ошибки в циклах for — распространённые проблемы, которые приводят к неправильной логике или бесконечному циклу.

Неверное условие выхода

Ошибка: если условие никогда не станет false, цикл будет выполняться бесконечно. 

// ❌ Бесконечный цикл
for (let i = 0; i < 5; i--) {
  console.log(i); // i будет уменьшаться: -1, -2, -3...
}
// ✅ Правильно
for (let i = 0; i < 5; i++) {
  console.log(i); // 0, 1, 2, 3, 4
}

Ошибка с индексом массива

// ❌ Неправильно
const arr = [1, 2, 3];
for (let i = 1; i <= arr.length; i++) {
  console.log(arr[i]); // Пропустит первый элемент, выведет undefined
}
// ✅ Правильно
const arr = [1, 2, 3];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]); // 1, 2, 3
}

Забывчивость про инкремент

// ❌ Бесконечный цикл
for (let i = 0; i < 5;) {
  console.log(i); // i никогда не изменяется
}
// ✅ Правильно
for (let i = 0; i < 5; i++) {
  console.log(i);
}

Лучшие практики использования for

Лучшие практики for — правила, которые делают циклы безопасными, понятными и эффективными.

  • Всегда используйте содержательные имена переменных (i, index для счётчиков).
  • Проверяйте условие выхода и убедитесь, что цикл завершится.
  • Для перебора массивов предпочитайте for...of или методы массива (forEach, map).
  • Избегайте изменения длины массива внутри цикла без необходимости.
  • Используйте break и continue для управления потоком цикла.

Использование break для ранего выхода

for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break; // Выход из цикла
  }
  console.log(i); // 0, 1, 2, 3, 4
}

Использование continue для пропуска итерации

for (let i = 0; i < 5; i++) {
  if (i === 2) {
    continue; // Пропустить эту итерацию
  }
  console.log(i); // 0, 1, 3, 4 (2 пропущено)
}

Современные альтернативы for

const arr = [1, 2, 3];

// for...of — более удобный способ
for (const item of arr) {
  console.log(item); // 1, 2, 3
}

// forEach — функциональный подход
arr.forEach((item, index) => {
  console.log(index, item);
});

Итоги

Итоги темы — основные правила работы с циклом for в JavaScript.

  • for используется, когда известно количество повторений.
  • Цикл состоит из инициализации, условия и инкремента.
  • Часто используется для перебора массивов и диапазонов чисел.
  • Необходимо проверять условие выхода, чтобы избежать бесконечного цикла.
  • break и continue помогают управлять потоком выполнения цикла.
  • Для современного кода рекомендуется for...of и методы массива.
