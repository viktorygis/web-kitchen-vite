📝 Практика по модулю - Система управления аккаунтами пользователя

Ты работаешь над проектом, который управляет аккаунтами пользователей. Тебе нужно будет заполнить пропуски в коде где будет:

  1. Проверять возраст пользователя.
  2. Определять, есть ли у пользователя подписка на сервис.
  3. Подсчитывать, сколько попыток входа совершил пользователь.

На основе этих данных программа будет принимать решения о том, какие действия могут быть выполнены.

1️⃣ Проверь возраста пользователя

Ты должен проверить, является ли пользователь совершеннолетним (возраст 18 или больше). Если это так, программа позволяет ему продолжить, а если нет — выводит сообщение о запрете.

Логика:

Если возраст больше или равен 18, вывести "Добро пожаловать!"

Если возраст меньше 18, вывести "Вам нельзя войти, возраст меньше 18."

2️⃣ Проверь подписку на сервис используя else if

Создай две переменные: hasSubscription, которая является true, а переменная с возрастом age пусть будет 20.

Мы будем проверять три варианта:

◽ Если у пользователя есть подписка И возраст равен или больше 18, вывести "Красава! Так держать"

◽ Если у пользователя нет подписки, вывести "Извини, тут только свои."

◽ Если возраст пользователя меньше 18 лет и у него нет подписки, вывести "Вам нужно купить подписку и подтвердить возраст".

3️⃣ Тернарный оператор

Теперь можно упростить логику проверки подписки с помощью тернарного оператора. Тернарный оператор позволяет сделать код короче как ты помнишь

◽ Создать переменную isSubscribed, которая будет хранить информацию о том, есть ли у пользователя подписка. Но у нее ее нет, по этому будет false

◽ Использовать тернарный оператор для проверки значения переменной isSubscribed:

Если isSubscribed равно true, выводим сообщение "Вы можете пользоваться сервисом!".

Если isSubscribed равно false, выводим сообщение "Пожалуйста, оформите подписку.".
