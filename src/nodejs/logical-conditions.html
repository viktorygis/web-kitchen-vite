Ветвления в программе

Как программа выбирает действия

Ветвление — это механизм, который позволяет программе выполнять разные действия в зависимости от условий.

Благодаря ветвлениям программа может принимать решения:

  • Если пользователь вошёл — открыть личный кабинет.
  • Если пароль неверный — показать ошибку.
  • Если возраст больше 18 — разрешить доступ.
  • Если товара нет — запретить покупку.

Условное ветвление if

if выполняет код только тогда, когда условие является истинным. 


const age = 20;

if (age >= 18) {
  console.log("Доступ разрешён");
}
  • В круглых скобках пишется условие.
  • Если условие true — выполняется код внутри блока.
  • Если условие false — блок пропускается.

Ветка else

else выполняется, если условие в if оказалось ложным. 


const passwordCorrect = false;

if (passwordCorrect) {
  console.log("Вход выполнен");
} else {
  console.log("Неверный пароль");
}

Несколько условий else if

Когда вариантов больше двух, используют конструкцию else if.

Каждое else if проверяется только если все предыдущие условия не сработали. 


const score = 85;

if (score >= 90) {
  console.log("Отлично");
}
else if (score >= 70) {
  console.log("Хорошо");
}
else if (score >= 50) {
  console.log("Удовлетворительно");
}
else {
  console.log("Нужно повторить");
}

Вложенные условия

Иногда одно условие проверяется только после выполнения другого. Такие конструкции называются вложенными. 


const user = {
  loggedIn: true,
  isAdmin: true
};

if (user.loggedIn) {

  if (user.isAdmin) {
    console.log("Панель администратора");
  }

}

Тернарный оператор

Тернарный оператор — короткая запись простого условия. 


const age = 18;

const result = age >= 18
  ? "Можно войти"
  : "Доступ запрещён";

console.log(result);

Комбинация условий

Ветвления часто используют вместе с логическими операторами.

Оператор Значение Пример
&& И age >= 18 && active
|| ИЛИ admin || moderator
! НЕ !blocked

Практические примеры

Ветвления используются практически во всех приложениях. 


function checkAccess(user) {

  if (!user) {
    return "Пользователь не найден";
  }

  if (user.role === "admin") {
    return "Полный доступ";
  }

  if (user.active) {
    return "Обычный доступ";
  }

  return "Доступ запрещён";
}

Итоги

  • Ветвления позволяют программе выбирать разные действия.
  • if выполняет код при истинном условии.
  • else используется для альтернативного варианта.
  • else if помогает создавать несколько вариантов выбора.
  • Вложенные условия позволяют строить сложную логику.
  • Тернарный оператор делает простые проверки короче.
