Использование классов в JavaScript

Класс — шаблон для создания объектов с общими свойствами и методами.
В Node.js классы используются для организации кода, создания моделей данных и построения архитектуры приложений.

Классы появились в JavaScript стандарта ES6 и являются удобным способом работы с объектно-ориентированным программированием. Под капотом они используют прототипное наследование JavaScript.

Создание класса

Объявление класса — создание структуры, которая описывает свойства и поведение будущих объектов.

  • class — ключевое слово для создания класса.
  • constructor() — специальный метод для начальной настройки объекта.
  • new — оператор создания экземпляра класса.

Простой класс пользователя

class User {

  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

}

const user = new User(
  "Alex",
  25
);

console.log(user.name);

Каждый объект, созданный через new User(), получает собственные свойства name и age.

Методы класса

Методы класса — функции внутри класса, которые описывают действия объекта.

Добавление методов

class User {

  constructor(name) {
    this.name = name;
  }


  sayHello() {
    console.log(
      `Привет, ${this.name}`
    );
  }

}


const user = new User("Alex");

user.sayHello();

Геттеры и сеттеры

get и set — специальные методы для безопасного чтения и изменения свойств объекта. 

class Product {

  constructor(price) {
    this._price = price;
  }


  get price() {
    return this._price;
  }


  set price(value) {
    this._price = value;
  }

}


const item = new Product(100);

item.price = 200;

console.log(item.price);

Наследование классов

Наследование — механизм, позволяющий создавать новый класс на основе существующего.

  • extends — создаёт связь между классами.
  • super() — вызывает конструктор родительского класса.

Пример наследования

class Animal {

  constructor(name) {
    this.name = name;
  }


  speak() {
    console.log(
      "Звук животного"
    );
  }

}


class Dog extends Animal {


  speak() {
    console.log(
      "Гав!"
    );
  }

}


const dog = new Dog("Bob");

dog.speak();

Статические методы

static — метод класса, который вызывается напрямую через класс, а не через объект. 

class MathHelper {


  static sum(a, b) {

    return a + b;

  }


}


console.log(
  MathHelper.sum(5, 10)
);

Классы в Node.js

Классы Node.js — удобный способ создавать структуры приложения: сервисы, модели, контроллеры и менеджеры данных.

Пример сервиса пользователя

class UserService {

    this.database = database;

Типичные ошибки:
  • Забывать использовать new при создании объекта.
  • Использовать стрелочные функции вместо методов класса без понимания контекста.
  • Изменять свойства объекта напрямую без проверки.

Лучшие практики работы с классами

  • Один класс должен отвечать за одну задачу.
  • Используйте понятные имена классов.
  • Не создавайте слишком большие классы.
  • Используйте наследование только когда оно действительно необходимо.

Итоги

  • Классы являются шаблонами для создания объектов.
  • constructor() задаёт начальное состояние объекта.
  • Методы описывают поведение объекта.
  • extends позволяет создавать дочерние классы.
  • Классы широко применяются в Node.js проектах.
