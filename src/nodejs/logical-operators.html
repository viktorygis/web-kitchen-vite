Логические операторы в условиях

Как программа проверяет несколько условий

Логические операторы позволяют объединять несколько условий и создавать более сложные проверки в программе.

Они помогают программе принимать решения на основе нескольких факторов:

  • Пользователь должен быть авторизован и активен.
  • Возраст должен быть больше 18 или иметь специальное разрешение.
  • Аккаунт не должен быть заблокирован.

Оператор && (И)

Оператор && возвращает true только тогда, когда оба условия являются истинными. 


const age = 25;
const hasTicket = true;

if (age >= 18 && hasTicket) {
  console.log("Можно войти");
}
  • Первое условие должно быть true.
  • Второе условие должно быть true.
  • Если хотя бы одно условие false — результат false.

Оператор || (ИЛИ)

Оператор || позволяет выполнить действие, если хотя бы одно из условий истинное. 


const isAdmin = false;
const isModerator = true;

if (isAdmin || isModerator) {
  console.log("Есть доступ");
}
  • Достаточно выполнения одного условия.
  • Если оба условия false — результат false.

Оператор ! (НЕ)

Оператор ! изменяет значение условия на противоположное. 


const blocked = false;

if (!blocked) {
  console.log("Пользователь активен");
}

Если значение было true, оператор ! сделает его false. Если значение было false — станет true.

Приоритет логических операторов

Когда в условии несколько операторов, они выполняются по определённому порядку: Первым у нас идет ! инверсия значения, после → && проверяет, что оба условия trueи на последнем месте || проверяет, что хотя бы одно условие true

Приоритет Оператор Описание
1 ! НЕ
2 && И
3 || ИЛИ 

true || false && false

// сначала выполнится &&
// затем ||

Комбинация условий

В реальных приложениях часто используются несколько операторов одновременно. 

const user = {
  age: 22,
  active: true,
  banned: false
};

if (
  user.age >= 18 &&
  user.active &&
  !user.banned
) {
  console.log("Доступ разрешён");
}

В сложных выражениях важно группировать условия в скобки () Сначала у нас проверяется условие в скобках. В нашем случае должно выдать true. И потом, чтобы мы могли пойти покупаться 🏊‍♂️, нужно, чтобы оба условия были true

const isSummer = true;
const isHot = false;
const isSwimmingPoolOpen = true;
if (isSummer&&(isHot||isSwimmingPoolOpen)) {
  console.log("Идём купаться! 🏊‍♂️");
}

Практические примеры

Логические операторы используются при авторизации, проверке прав доступа и обработке данных. 


function canBuyProduct(user, product) {

  if (
    user.loggedIn &&
    user.balance >= product.price &&
    product.available
  ) {
    return "Покупка разрешена";
  }

  return "Покупка невозможна";
}

Итоги

  • Оператор && объединяет условия через "И".
  • Оператор || проверяет вариант "ИЛИ".
  • Оператор ! меняет значение условия на противоположное.
  • Логические операторы позволяют создавать сложные проверки.
  • Они являются основой условий в JavaScript и Node.js.
