Цикл while JavaScript

Повторение действий пока выполняется условие

Цикл while — это конструкция JavaScript, которая выполняет блок кода повторно, пока указанное условие возвращает значение true.
Количество повторений заранее неизвестно и зависит от изменения состояния программы.

Циклы используются, когда программе необходимо выполнять одинаковые действия несколько раз. Например, обработать данные, ожидать событие или выполнять проверку до получения нужного результата.

Где используется while

  • Ожидание выполнения условия.
  • Обработка данных неизвестного количества.
  • Работа с пользовательским вводом.
  • Создание повторяющихся процессов.

Что такое цикл while

while — это цикл с предварительной проверкой условия.
Перед каждым выполнением блока кода JavaScript сначала проверяет выражение внутри скобок.

Если условие возвращает true, тело цикла выполняется. Если условие становится false, цикл завершается.

Простая схема работы while:

  • Проверить условие.
  • Если условие истинно — выполнить код.
  • Изменить данные внутри цикла.
  • Повторить проверку.

Синтаксис цикла while

Синтаксис while — это структура записи цикла, состоящая из ключевого слова while, условия и блока выполняемого кода.

Базовая структура:

while (условие) {
  // код выполняется пока условие true
}
  • Условие находится внутри круглых скобок.
  • Код цикла располагается внутри фигурных скобок.
  • Условие должно когда-нибудь стать false.

Пример:

let count = 0;
while (count < 3) {
  console.log("Привет! 😊");
  count++;
}
Как это работает ⚒
  1. count = 0, проверяется count < 3 → условие true, запускаем код.
  2. Выводится "Привет! 😊", count++ увеличивает count на 1.
  3. Цикл повторяется, пока count < 3
  4. Когда count = 3, условие false, и цикл останавливается.

Как работает цикл while

Принцип работы while — последовательная проверка условия перед каждым повторением.
Цикл полностью зависит от изменения переменной или состояния внутри блока.

Пример счётчика:

let count = 1;

while (count <= 5) {
  console.log(count);
  count++;
}

В этом примере цикл выполнится пять раз. После каждого выполнения переменная count увеличивается.

Шаг Значение count Результат проверки
1 1 true
2 2 true
6 6 false — остановка

Условие остановки цикла

Условие остановки — это изменение данных, которое приводит выражение while к значению false.
Без него цикл может выполняться бесконечно.

Корректное изменение условия:

let attempts = 0;

while (attempts < 3) {
  console.log("Попытка");
  attempts++;
}

Важно: если переменная условия никогда не изменяется, цикл продолжит выполняться бесконечно.

Ошибка: отсутствие изменения переменной

let num = 1;
while (num < 5) {
  console.log(num);
} // ⚠️ num не увеличивается, цикл никогда не остановится!

Правильный вариант:

let num = 1;
while (num < 5) {
  console.log(num);
  num++; // ✅ num увеличивается, цикл завершится при num = 5
}

Математические операции с использованием while

Циклы полезны не только для вывода сообщений, но и для выполнения математических операций, например, сложения чисел.

Пример: суммирование чисел от 1 до 5

let sum = 0;   // Переменная, в которой будем накапливать сумму
let num = 1;   // Начинаем с 1

while (num <= 5) { // Пока num меньше или равно 5, продолжаем цикл
  sum += num;  // Прибавляем num к sum
  num++;       // Увеличиваем num на 1
}

console.log("Сумма:", sum);

В этом примере цикл while суммирует числа от 1 до 5. После завершения цикла переменная sum будет равна 15.

Перед началом цикла:
◽ sum = 0 (пустая копилка, куда будем складывать числа).
◽ num = 1(начинаем складывать с 1).

Первый проход num = 1 :
◽ sum = sum + num → 0 + 1 = 1.
◽ num++, теперь num = 2

Второй проход num = 2 :
◽ sum = sum + num → 1 + 2 = 3.
◽ num++, теперь num = 3
и так далее...

Пока не будет num = 6, а условие num <= 5 ложное false, поэтому цикл завершается. В sum теперь хранится 15, и программа выводит "Сумма: 15"

Другие примеры математических операций с while:

Умножение чисел: вычисление факториала.

Факториал числа n (обозначается n!) — это произведение всех положительных целых чисел от 1 до n. Например, 5! = 5 × 4 × 3 × 2 × 1 = 120.

let factorial = 1;
let n = 5;
while (n > 1) {
  factorial *= n;
  n--;
}
console.log("Факториал:", factorial);

Нахождение максимального или минимального значения в массиве.

Этот пример показывает, как с помощью цикла while найти максимальное значение в массиве чисел.

const numbers = [3, 7, 2, 9, 4];
let max = numbers[0];
let index = 1;
while (index < numbers.length) {
  if (numbers[index] > max) {
    max = numbers[index];
  }
  index++;
}
console.log("Максимальное значение:", max);

Нахождение среднего значения: суммирование и деление на количество элементов.

Этот пример демонстрирует, как с помощью цикла while вычислить среднее значение элементов массива.

const values = [10, 20, 30, 40];
let total = 0;
let count = 0;
while (count < values.length) {
  total += values[count];
  count++;
}
console.log("Среднее значение:", total / values.length);

Подсчёт количества чётных чисел

В этом примере мы используем цикл while для подсчёта количества чётных чисел. Используется оператор % - остаток от деления для проверки чётности.

let num = 1;
while(num <= 6) {
  if (num %2===0) {
    console.log(num);
  }
  num++;
}

На каждой итерации проверяем условие:
◽ Если num % 2 === 0, значит число чётное (остаток от деления на 2 равен 0).
◽ Если условие истинно, выводим число в консоль.
В конце каждой итерации обязательно увеличиваем num , чтобы избежать бесконечного цикла и перейти к следующему числу.

Когда использовать while

Когда использовать while — цикл подходит для ситуаций, когда заранее неизвестно количество повторений.
Например, при обработке данных, ожидании события или повторной проверке состояния программы.

  • Ожидание пользовательского ввода.
  • Обработка данных из потока.
  • Повторная проверка состояния приложения.
  • Выполнение действий до достижения определённого условия.

При использовании while

Мы не знаем заранее, сколько раз temperature-- уменьшит значение, пока условие не станет false

let temperature = 35;
while (temperature > 30) {
  console.log(`Температура: ${temperature}`);
  temperature--;
}
console.log("Стало прохладнее!");

В этом примере цикл while выполняется, пока температура выше 30 градусов. После каждого прохода температура уменьшается на 1, и когда она достигает 30, цикл останавливается.

while и for

Сравнение while и for — выбор подходящего цикла зависит от того, известно ли заранее количество повторений.
Оба цикла выполняют повторяющийся код, но используются в разных ситуациях.

Цикл Когда использовать Пример
while Количество повторений неизвестно заранее. Ожидание условия, обработка данных.
for Количество повторений известно заранее. Перебор массива, счётчик.

Пример for:

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

Пример while:

let i = 1;

while (i <= 5) {
  console.log(i);
  i++;
}

Если количество повторений зависит от внешнего условия, чаще подходит while. Если нужен перебор известного количества элементов — обычно выбирают for.

Практические примеры while

Практическое применение while — выполнение повторяющихся действий, когда заранее неизвестно количество операций.
Цикл часто используется при работе с потоками данных и ожиданием событий.

Обработка массива данных

Перебор элементов массива:

const users = [
  "Anna",
  "Ivan",
  "Maria"
];

let index = 0;

while (index < users.length) {
  console.log(users[index]);
  index++;
}

Такой подход позволяет вручную управлять индексом и условием остановки.

Пример в Node.js

while в Node.js — цикл может использоваться для обработки очередей задач или повторных проверок состояния приложения.

Обработка очереди задач:

const tasks = [
  "build",
  "test",
  "deploy"
];

while (tasks.length > 0) {
  const task = tasks.shift();
  console.log("Выполняется:", task);
}

После выполнения каждой задачи элемент удаляется из массива. Когда очередь становится пустой, цикл завершается.

Бесконечные циклы

Бесконечный цикл — это цикл, условие которого никогда не становится false.
Такая ситуация приводит к постоянному выполнению кода.

Ошибка с отсутствием изменения переменной:

let count = 0;

while (count < 5) {
  console.log(count);
}

Причина ошибки — переменная count не изменяется внутри цикла, поэтому условие всегда остаётся true.

Правильный вариант:

let count = 0;

while (count < 5) {
  console.log(count);
  count++;
}

Типичные ошибки при работе с while

Ошибки while — распространённые проблемы, которые приводят к неправильной логике или зависанию программы.

Ошибка Причина Решение
Нет изменения условия Переменная остаётся прежней. Изменять значение внутри цикла.
Неверное условие Цикл завершается слишком рано. Проверить логику сравнения.
Слишком сложный цикл Много действий внутри блока. Разделить код на функции.

Лучшие практики использования while

Лучшие практики while — правила, которые делают циклы безопасными и понятными.

  • Всегда проверяйте, изменяется ли условие завершения.
  • Используйте понятные имена переменных.
  • Не помещайте слишком много логики внутрь цикла.
  • Используйте другие циклы, если они подходят лучше.
  • Добавляйте защиту от бесконечного выполнения.
