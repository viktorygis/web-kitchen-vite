Тернарный оператор

Короткая запись условий

Тернарный оператор — это специальный оператор, который позволяет записать простое условие в одну строку.

Он является сокращённой формой конструкции if / else.

  • Проверяет условие.
  • Возвращает одно значение, если условие true.
  • Возвращает другое значение, если условие false.

Синтаксис тернарного оператора

Тернарный оператор ? : позволяет записывать простые условия в одну строку. Он заменяет if...else, если нужно выбрать одно из двух значений.

условие ? значение_если_true : значение_если_false;

Тернарный оператор состоит из трёх частей:

  • Условие — что нужно проверить.
  • Знак ? разделяет условие и первый результат.
  • Знак : разделяет два возможных значения.

Пример:

let result = 5 > 3 ? "Верно!" : "Неверно!";

Обычный if...else :

let result;
if (5 > 3) {
  result = "Верно!";
} else {
  result = "Неверно!";
}

Принимает значение

Тернарный оператор возвращает значение, которое можно присвоить переменной или использовать в выражении.

const age = 20;
const message = age >= 18
  ? "Совершеннолетний"
  : "Несовершеннолетний";
console.log(message); // "Совершеннолетний"

Можно сразу выводить результат через console.log() 

const age = 20;
console.log(age >= 18 ? "Доступ разрешён" : "Доступ запрещён");

Использование в шаблонных строках

Тернарный оператор удобно использовать внутри шаблонных строк.

Шаблонные строки template literals удобный способ работать со строками

Они позволяют:

  • ✅ Вставлять переменные прямо внутрь строки
  • ✅ Писать длинные строки на нескольких строках без +
  • ✅ Использовать выражения внутри строки

Обычные строки пишутся в кавычках " ", а шаблонные строки используют обратные кавычки ` а вместо + для соединения строк мы просто вставляем переменные в ${ } 

const message = `Привет, ${name}!`;
const isOpen = false;
console.log(`Парк${isOpen ? "открыт" : "закрыт"}!`);

В зависимости от значения isOpen будет выведено "Парк открыт!" или "Парк закрыт!".

Простые условия

const age = 20;

const message = age >= 18
  ? "Совершеннолетний"
  : "Несовершеннолетний";

console.log(message);

Если условие age >= 18 истинное, будет выбран первый вариант.

Сравнение с if / else

Один и тот же код можно написать двумя способами.

Через if / else:


let access;

if (isAdmin) {
  access = "Разрешено";
} else {
  access = "Запрещено";
}

Через тернарный оператор:


const access = isAdmin
  ? "Разрешено"
  : "Запрещено";

Вложенный тернарный оператор

Несколько тернарных операторов можно объединять, но такой код может стать сложным для чтения. 


const score = 85;


const result =
  score >= 90
    ? "Отлично"
    : score >= 60
      ? "Хорошо"
      : "Нужно повторить";


console.log(result);

Для большого количества условий лучше использовать if / else if / else.

Тернарный оператор в функциях

Часто используется для возврата простого результата из функции. 


function checkAge(age) {

  return age >= 18
    ? "Доступ разрешён"
    : "Доступ запрещён";

}

Частые ошибки

  • Использование тернарного оператора вместо сложной логики.
  • Создание слишком длинных вложенных условий.
  • Попытка заменить им все конструкции if.
  • Плохая читаемость большого количества условий.

Практические примеры

Тернарный оператор часто используется при отображении данных. 


const user = {
  name: "Иван",
  online: true
};


const status = user.online
  ? "Онлайн"
  : "Оффлайн";


console.log(status);



const price = 1000;

const discount =
  price > 500
    ? price * 0.9
    : price;


console.log(discount);

Итоги

  • Тернарный оператор является короткой формой if / else.
  • Он использует три части: условие, true-значение и false-значение.
  • Подходит для простых проверок и присваивания значений.
  • Сложные условия лучше писать через обычные ветвления.
  • Тернарный оператор часто используется в JavaScript и Node.js.
