Атрибуты HTML

Атрибуты HTML — это специальные параметры, которые задаются внутри тега и позволяют дополнительно настраивать его работу, внешний вид или поведение.

  • Атрибуты всегда указываются внутри открывающего тега: <тег атрибут="значение"> .
  • С помощью атрибутов можно задать идентификатор, класс, стили, ссылки, параметры для изображений, форм, списков и других элементов.
  • Некоторые атрибуты универсальны для большинства тегов, другие — специфичны для определённых элементов.
<a href="https://example.com" target="_blank">Ссылка</a>
<img src="photo.jpg" alt="Фото">
<input type="text" placeholder="Введите текст">
<div class="container" id="main">...</div>

Типы и примеры атрибутов

  • class

    class — назначает CSS-класс для стилизации элемента.

    • Может использоваться для группировки элементов с одинаковым стилем.
    • Допустимо указывать несколько классов через пробел.

    Пример:

    <div class="box highlight">Текст</div>

  • id

    id — уникальный идентификатор элемента на странице.

    • Используется для обращения к элементу через CSS или JavaScript.
    • На странице не должно быть двух элементов с одинаковым id .

    Пример:

    <h2 id="intro">Введение</h2>

  • style

    style — позволяет задать встроенные CSS-стили прямо в элементе.

    • Удобно для быстрого изменения оформления, но для сложных стилей лучше использовать CSS-файл.

    Пример:

    <p style="color: red; font-weight: bold;">Красный текст</p>

  • title

    title — текстовая подсказка, отображается при наведении курсора на элемент.

    • Работает для большинства тегов, улучшает доступность.

    Пример:

    <span title="Подробнее">?

  • data-*

    data-* — пользовательские атрибуты для хранения дополнительных данных.

    • Позволяет хранить любые значения для работы скриптов или стилей.
    • Пример: data-user="123" , data-theme="dark"

    Пример:

    <div data-user="123" data-theme="dark">...

  • src , href , alt , target

    src — путь к файлу (например, изображение, видео, скрипт).
    href — адрес ссылки.
    alt — альтернативный текст для изображения.
    target — способ открытия ссылки: _blank (в новой вкладке), _self (в той же).

    • Часто используются для работы с мультимедиа и навигацией.

    Примеры:

    <img src="cat.jpg" alt="Кот">
<a href="https://example.com" target="_blank">Ссылка</a>

  • disabled , checked , selected , readonly

    disabled — делает элемент неактивным.
    checked — отмечает чекбокс или радиокнопку.
    selected — выбирает элемент списка.
    readonly — запрещает редактирование поля.

    • Используются в формах и интерактивных элементах.

    Примеры:

    <input type="checkbox" checked>
<input type="text" disabled>
<select>
  <option selected>Первый</option>
</select>

  • placeholder , value , name

    placeholder — текст-подсказка внутри поля ввода.
    value — текущее значение элемента.
    name — имя поля для передачи данных формы.

    • Важны при создании форм для ввода и отправки информации.

    Пример:

    <input type="text" name="user" value="Вася" placeholder="Введите имя">
