Теги форматирования текста

Теги форматирования текста — это специальные элементы HTML, которые позволяют изменять внешний вид текста на веб-странице. Они помогают выделять важные части текста, добавлять стили и улучшать читаемость контента.

Форматирование текста включает в себя такие элементы, как жирный шрифт, курсив, подчёркивание и другие. Эти теги не только улучшают визуальное восприятие, но и помогают передать смысловую нагрузку текста.

Виды тегов форматирования текста

  • <b>

    <b> — Выделяет текст жирным шрифтом без придания ему дополнительной семантической важности.

    Использование: Для визуального выделения слов, которые не несут специального смысла (например, в рекламе, интерфейсах).

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Для смыслового выделения используйте <strong> .

    Пример:

    <p>Это <b>важное</b> слово выделено жирным.</p>

    Пример: Жирный текст .

  • <strong>

    <strong> — Обозначает текст с высокой важностью или предупреждением, обычно отображается жирным.

    Использование: Для акцентирования внимания на важных предупреждениях, инструкциях, действиях.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Семантически важнее, чем <b> — влияет на SEO и доступность.

    Пример:

    <strong>Внимание!</strong> Это важное предупреждение.

    Пример: Важный текст .

  • <i>

    <i> — Отображает текст курсивом для выделения альтернативного голоса, термина, мысли.

    Использование: Для иностранных слов, научных терминов, цитат, названий.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Семантика меньше, чем у <em> .

    Пример:

    <p>Термин <i>Homo sapiens</i> на латыни.</p>

    Пример: Курсивный текст .

  • <em>

    <em> — Подчеркивает смысловой акцент на тексте, обычно отображается курсивом.

    Использование: Для выражения интонации, акцента, особого отношения.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Семантически выделяет важный акцент для скринридеров.

    Пример:

    Я <em>настоятельно</em> рекомендую это сделать.

    Пример: Акцентируемый текст .

  • <small>

    <small> — Представляет побочные комментарии, сноски, юридические примечания мелким шрифтом.

    Использование: Для копирайта, условий, пояснений.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Не используйте для обычного текста — только для неосновной информации.

    Пример:

    <small>© 2023 Компания. Все права защищены.</small>

    Пример: Уменьшенный текст .

  • <mark>

    <mark> — Выделяет текст как маркером, обычно жёлтой подсветкой (например, в результатах поиска).

    Использование: Для акцента на важных или найденных фрагментах текста.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Цвет подсветки можно менять через CSS.

    Пример:

    В тексте найдено: <mark>ключевое слово</mark>.

    Пример: Выделенный текст .

  • <del>

    <del> — Показывает удаленный фрагмент текста (перечеркнутый), например, старую цену.

    Использование: Для редактирования, истории изменений, отменённых пунктов.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Часто используется вместе с <ins> для показа изменений.

    Пример:

    Цена: <del>5000 руб.</del> 3500 руб.

    Пример: Удаленный текст .

  • <ins>

    <ins> — Показывает вставленный фрагмент текста (подчеркнутый), например, новый вариант.

    Использование: Для показа изменений, новых пунктов, редакций.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Для визуального сравнения используйте с <del> .

    Пример:

    Версия 2.<ins>1</ins> с исправлениями.

    Пример: Вставленный текст .

  • <sub>

    <sub> — Нижний индекс для химических формул, математических обозначений, примечаний.

    Использование: Химия ( H 2 O ), математика, примечания.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Для верхнего индекса используйте <sup> .

    Пример:

    Вода: H<sub>2</sub>O

    Пример: H 2 O.

  • <sup>

    <sup> — Верхний индекс для степеней, порядковых числительных, сносок.

    Использование: Математика ( x 2 ), физика, сноски.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Для нижнего индекса используйте <sub> .

    Пример:

    Теорема Пифагора: a<sup>2</sup> + b<sup>2</sup> = c<sup>2</sup>

    Пример: x 2 .

  • <u>

    <u> — Подчеркивает текст, например для выделения орфографических ошибок, имён собственных.

    Использование: Для акцента, стилистических задач, выделения.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Не рекомендуется для гиперссылок — чтобы не путать с <a> .

    Пример:

    Это <u>подчеркнутое</u> слово.

    Пример: Подчеркнутый текст .

  • <br>

    <br> — Перенос строки внутри текста (адреса, стихи, подписи).

    Использование: Для разметки поэтических или адресных строк.

    • clear — устаревший атрибут для управления обтеканием (почти не используется).

    Примечание: Тег одиночный, не требует закрытия.

    Пример:

    Улица Пушкина,<br>дом Колотушкина

    Пример: Первая строка.
    Вторая строка.

  • <code>

    <code> — Обозначает фрагмент компьютерного кода с моноширинным шрифтом.

    Использование: Для выделения команд, переменных, функций, частей программ.

    • class — CSS-класс для стилизации.
    • id — уникальный идентификатор.
    • style — встроенные CSS-стили.

    Примечание: Внутри <pre> позволяет показывать форматированный код.

    Пример:

    Используйте <code>console.log()</code> для отладки.

    Пример: фрагмент кода .
