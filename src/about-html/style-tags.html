Теги стилей HTML

Теги стилей HTML — style и link это основные теги, используемые для добавления стилей и подключения внешних ресурсов в HTML-документ.
Они позволяют управлять внешним видом страницы, подключать CSS-файлы и другие ресурсы, такие как шрифты и иконки.

<style>

<style> — Позволяет встраивать CSS-стили непосредственно в HTML-документ. Содержимое тега интерпретируется браузером как CSS-код.

Где использовать: В разделе <head> или внутри элементов (с атрибутом scoped , но устарел).

  • type — MIME-тип (по умолчанию text/css ).
  • media — Определяет медиа-запрос для применения стилей.
  • scoped — Ограничивает стили родительским элементом (устарело).

Примечание: Для больших проектов рекомендуется использовать внешние стили через <link> .

Пример:

<style>
  body {
    background-color: #f0f0f0;
    font-family: Arial;
  }
</style>

<link>

  • <link> — Подключает внешние ресурсы к документу. Чаще всего используется для подключения CSS-файлов.

    Где использовать: В разделе <head> документа. Может подключать иконки, шрифты, стили.

    • rel — Тип отношения (обязательный): stylesheet , icon , preload .
    • href — Путь к ресурсу (обязательный).
    • type — MIME-тип ресурса.
    • media — Медиа-запрос для применения стилей.

    Примечание: Может использоваться для предзагрузки ресурсов ( rel="preload" ) для оптимизации скорости загрузки.

    Пример:

    <!-- Подключение CSS -->
<link rel="stylesheet" href="styles.css">

<!-- Иконка сайта -->
<link rel="icon" href="favicon.ico">
